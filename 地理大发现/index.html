地理大发现

最近编辑于 Jan 3, 2024

    大航海，也可以称为“地理大发现”。它让欧亚大陆的人们发现了广袤的新大陆，导致了随后的移民热潮，开启了真正的全球贸易时代。

    # 起因

    航海的目的是为了开辟贸易路线，和东方做生意，特别是进行香料贸易。

    但是，十五世纪，奥斯曼帝国崛起的同时，也在无形中不断破坏东西方的贸易网络。而在此之前，无论是阿拉伯人还是蒙古人，在军事征服的同时都会在一定程度上保护贸易

    在这样的背景下，西欧甚至南欧的国家，都不得不寻找新的航线，希望能绕过奥斯曼人的关卡，直接通往印度和中国做生意。

    当时，人们已经普遍接受托勒密地理学的观点，认为地球是圆的。既然东方被奥斯曼人控制了，他们就试图往南方、西方，甚至北方寻找新航道。最终，迪亚士和达伽马向南航行，哥伦布和麦哲伦向西航行，约翰∙卡伯特和巴伦支向北航行。

    # 葡萄牙的大航海

    首先，为什么欧洲的大航海起源于伊比利亚半岛，特别是葡萄牙呢？这主要是因为那里对新航线的需求最强，而且它们首先具备了开展大航海运动的条件。

    • 葡萄牙地处欧洲大陆最西端，在东西方贸易中被中间商层层盘剥，国家的财富都用来支付沿途的税费了。
    • 15世纪时，伊比利亚半岛上的人们将阿拉伯人的势力赶回了北非，在此过程中王权得到了加强，这让葡萄牙和西班牙能够举国家之力投资寻找新航线的活动。

    早期，王室作为大航海的风险投资人，随后，人们看到王室发财后，其他国家的人再进行大航海时，就有商人和市民们愿意集资交给航海家们去冒险挣钱了。

    在大航海中，为首的通常是那些极具冒险精神、渴望发财的探险家，比如达伽马和哥伦布等人，随行的水手们则主要是囚犯、以及一些破产者。因为那时候航行是件很危险的事，不仅不知道航行的路线，甚至连目的地在哪都不知道。也就是说，大家参与大航海并不是全部出于高尚的目的，更多的是对利润的贪婪。因此，我们也就不难理解为什么这些人到了新大陆后会如此野蛮地掠夺资源。

    # 为什么探险家们愿意参与其中

    葡萄牙王室采取的一系列鼓励措施。当时王室会颁发一种特许状给探险者，保证如果他们发现了新的海外领土，只要宣告这片领土的主权归葡萄牙所有，发现者就能成为新的领土的总督，有在那片土地上征税的特权。

    前面说到葡萄牙王室投资大航海，其中亨利王子值得一提

    1487年，我们熟悉的航海家迪亚士，受葡萄牙国王若奥二世委托，往南航行寻找非洲大陆的最南端。他率船队从葡萄牙里斯本出发，沿着西非海岸南下，最后到达了非洲的最南端，他为那里起名“风暴角”，以纪念航程中遇到的风暴。后来被葡萄牙国王改名为“好望角”，寓意为这个海角代表着成功开辟东方航线的美好希望

    迪亚士的这次探险的意义重大，这意味着欧洲人可以绕过陆地上奥斯曼帝国控制的地区，从海上航线与印度和亚洲其他地区展开贸易

    迪亚士的探索止于好望角，而完成绕过好望角、到达印度的壮举的，是另一个我们熟悉的航海家达伽马。

    1502年，葡萄牙国王让达伽马率领一支庞大舰队再度出发前往印度，准备用军事力量来巩固新航线。抵达印度后，达伽马的舰队征服了该地，并且以进行贸易作为和平的条件，就这样建立起了葡萄牙和印度的贸易关系。达伽马这次返回葡萄牙时，载满了贵重的印度货物，主要是香料。

    在随后的一个多世纪里，葡萄牙通过与东方进行贸易，一度成为了欧洲最富有的国家。

    # 西班牙

    在葡萄牙已经证明了可以向南绕过非洲前往印度的情况下，西班牙又为何会选择向西走上一条截然不同的道路，从而来到美洲呢？

    15世纪，当葡萄牙战胜摩尔人并逐渐崛起的同时，伊比利亚半岛上的另外两个国家——卡斯蒂亚联合王国和阿拉贡联合王国也崛起了。1469年，卡斯蒂亚的公主伊莎贝拉与阿拉贡的王子费迪南结婚，两人的联姻使这两个国家合二为一，形成了西班牙王国。

    西班牙统一之后，综合国力得到了提高。当时欧洲各个王国都企图通过建立贸易航线来扩充财富，西班牙王国也不甘落后。但是，东西方贸易非常重要的一条通道，也就是从欧洲往南途经西非和大西洋的航道，都已经被葡萄牙人把持了。这条航道上一路都是葡萄牙人建立的殖民点和中转站。

    在葡萄牙人不愿意把港口给西班牙人使用的情况下，热那亚的意大利水手哥伦布找到了西班牙国王，声称他能找到通往印度的新航线。

    虽然哥伦布当时不曾给西班牙带来大量的财富。但是根据当时谁先发现的地方就属于谁的原则，西班牙获得了几乎全部的美洲。后来在那里发现的银矿和金矿，让西班牙很快超过葡萄牙成为欧洲最富有的国家。

    # 地理大发现的影响

    # 旧大陆带给新大陆的

    • 欧洲人除了自己移民到美洲外，首先带到美洲的是非洲裔的人口。今天，大约1亿非洲裔生活在巴西，超过4600万生活在美国。

    • 欧洲的马被引进到美洲后，极大地提高了那里的生产力，并且成为了主要的交通工具。

    • 在哥伦布到达美洲之前，美洲并没有畜牧业，欧洲人把牛羊带入之后，美洲部分地区后来成为了全世界畜牧业最发达的地区。

    • 无意间带去了蚯蚓。

      在哥伦布大交换前的北美，因为没有蚯蚓，土壤经过数年的耕种和浇灌，就会板结，而且肥力下降很快。这也是早期印第安人在一个地区耕种一段时间后，就不得不抛荒迁徙的一个重要原因。蚯蚓的引入，使得美洲农作物的单产得到了提升。

      除了影响农业生产，蚯蚓还改变了美洲大陆的生态环境。在蚯蚓到达美洲之前，在森林里，大量的枝叶腐烂，会让昆虫大量繁殖，然后吸引来鸟类吃昆虫。蚯蚓到达美洲之后，枯枝败叶很快被蚯蚓吃掉，然后把肥料输入回土壤中，植被变得茂密，但是昆虫和鸟类数量却减少了

      • 还从旧大陆带去了超过50种植物品种，其中咖啡、甘蔗和香蕉这三种经济作物，在美洲新大陆被广泛种植。它们支撑起了新大陆的种植业。
      • 欧洲人把天花等疾病带到美洲，让大量没有抵抗力的原住民死亡；

    # 新大陆带给旧大陆的

    • 从美洲带回了玉米、番薯、马铃薯这几种产量极高的物种。这成为了后来欧亚大陆重要的粮食作物，助推欧亚大陆的人口突破了自古典文明以来的瓶颈。
    • 蔬菜水果，比如西红柿、辣椒、四季豆、花生、菠萝、南瓜、草莓、等等，还有很多人爱吃的小龙虾（克氏原螯虾）。不过要注意的是，其中有些植物虽然是新旧大陆都有，但是美洲的是良种，它们的传入也极大地影响了旧大陆。
    • 也让烟草这种非常害人的植物，在世界范围内传播。
    • 殖民者也从美洲带回了传染病，让旧大陆的人深受其害，其中危害最大的就是梅毒。

    # 发现新大陆后

    伊比利亚半岛上的两个王国就为将来各自的贸易范围争了起来。有意思的是，当时它们两国的船队都还没有到达亚洲。

    1493年，教宗像切西瓜似的，把地球一分为二。分界线是位于非洲西海岸佛得角群岛以西100里格的那条经线，里格是当时绘制地图的度量单位。这条线以西的未知土地都归西班牙，以东的未知土地都归葡萄牙。

    这次划分暂时止住了两个国家的争端，西班牙和葡萄牙开始专注于发展自己的殖民地。不过，西班牙人当时虽然占据了美洲，但是直到他们在那里发现金银矿之前，其实没有获得什么实际的好处。而葡萄牙则通过香料贸易获得了巨大利润。

    在经过了一系列的失败后，西班牙终于迎来了一位伟大的航海家麦哲伦，帮他们找到了绕过美洲到达亚洲的航道。

    在巨大的商业利润的驱使下，西欧各国开始效仿葡萄牙和西班牙，出海做贸易，开辟新航线，这些探险家们的战舰和商船发现了地球上越来越多的大陆和岛屿。 其中，🇳🇱 荷兰和🇬🇧 英国也取得了瞩目的成就

    1496年，也就是哥伦布发现新大陆的第四年，英王亨利七世就雇了威尼斯航海家卡波特（John Cabot，1450 – 1499）去寻找新大陆。在此后的几十年里，英国的航海家到达了北美洲的纽芬兰，哈德逊湾和弗吉尼亚等地。

    # 贸易的利润

    # 葡萄牙

    葡萄牙人自从占据了印度和东南亚的香料群岛以后，就基本上控制了欧洲的香料生意。

    在此之前，虽然中国的商品也可以通过阿拉伯商人中转卖到欧洲，但是贸易的数量总不如欧洲人直接过来买的多。中国的瓷器、丝绸和茶叶也是从那时开始，经过葡萄牙商人大量进入欧洲的。

    在当时，东西方贸易，特别是瓷器贸易的利润高得难以置信，甚至高达成本的几百倍。

    两个例子：
    一、有历史学家通过计算认为，达伽马首航的收益就相当于成本的60倍。要知道，达伽马首航的船队只有三条船，就能取得如此大的利润，在之后更大规模的贸易中，产生的利润肯定要比第一次还要高得多。

    二、 几个世纪后的哥德堡号沉船事件。1745年，瑞典的哥德堡号（Ship Gothenburg）满载着700吨从中国采购的货物——主要是茶叶、瓷器和丝绸，从广州启程回国，九月份驶进了哥德堡海港。在近海触礁导致茶叶和丝绸都无法售卖了，靠仅剩的一点瓷器仍覆盖了两年的成本，并让股东们获得 10%+的利润。

    # 西班牙

    过去有一种观点认为，欧洲人近代的财富（或者至少是资本原始积累）是靠掠夺殖民地得来的。这个说法不完全准确。

    因为和整个殖民时期欧洲人创造和获得的总财富相比，直接掠夺而来的财富只占非常小的比例。 原因很简单，新发现的大陆的原住民文明程度很低，没有多少财富可以掠夺。

    西班牙人获取财富的几个主要渠道：

    1. 以印加帝国国王为人资，掠夺了帝国大量的黄金，据说这笔黄金有一万三千磅，大约六吨。但跟航海贸易利润比起来就不突出了，并且是一锤子买卖
    2. 1545年，西班牙人在玻利维亚发现了大银矿。第二年他们又在墨西哥的萨卡特卡斯（Zacatecas）发现了一处更大的银矿。在西班牙发现银矿后的一个多世纪里，他们从那里带走了数百吨的黄金和数万吨白银。靠着这些金银，西班牙取代了葡萄牙成为了当时欧洲最富有、最强大的国家。
      为了开采金银矿，西班牙人将非洲的黑人贩卖到美洲做奴隶，三角贸易的雏形出现了。
    3. 1519年，西班牙国王查理五世(Charlie I，1500～1558)在他祖父去世后，从他祖父手里继任神圣罗马帝国皇帝。作为当时神圣罗马帝国的一部分，尼德兰地区也就成为了西班牙的一部分。当时，尼德兰地区是欧洲手工业最发达的地区，后来也加入到全球贸易当中。这部分税收，后来一度占到西班牙国库收入的一半。

    1580年，西班牙兼并了葡萄牙，从此西班牙的殖民地和贸易港口遍及全世界。西班牙也因此成为了世界历史上第一个“日不落帝国”。

    大航海时期的西班牙不仅富有，而且强大。为了保证海上贸易，西班牙还用黄金白银建立起一支强大的海军——无敌舰队

    # 西班牙的衰落

    那么曾经如此富有，如此辉煌的西班牙为什么在之后会被比它晚开始航海的英国赶超呢？简单来说，财富是一把双刃剑，用得好可以促进文明的发展，用不好则会让一个国家和民族堕落。

    当财富大量涌入西班牙后，原本已经萌芽的资本主义工商业不仅没有得到发展，反而被扼杀了。制造业并没有得到发展。

    到了十七世纪，当荷兰人经过80年的抗争从西班牙独立后，西班牙的收入来源又少了一大笔。
