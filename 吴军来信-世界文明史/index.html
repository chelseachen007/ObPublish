    # 中世纪

    # 东方

    古典文明在东汉末年终结，从那以后，中原王朝就陷入了分裂时期。西晋虽然短暂地统一过中原，但很快被推翻了。

    中原王朝的再次统一，要等到公元589年，由隋王朝来完成，东亚也进入到了中世纪的第二个阶段。无论是隋朝还是后来的唐朝，都将整个中原农耕地区和草原游牧地区统一了起来，这在中国历史上是首次

    • 海纳百川，接纳朝鲜、日本、越南、吐蕃、印度、西域等国官员和将领，士人、商人、僧侣来交流学习和做生意。
    • 在唐代，很多读书人学习梵语和印度的佛学，促进了儒释道三教的融合。
    • 隋朝之前，那种贵族和豪族干预和控制政治的情况再也没有发生，从贵族门阀政治，走向了君主集权。
    • 从这个时期开始，中国确定了后来沿用了上千年的文官制度，以及通过科举考试选拔人才的制度。
    • 最重要的是，自唐中叶之后，中国自古以来的经济重心开始南移，从此确立了南方的经济地位。可以讲，隋唐宋在中华文明的发展脉络上划开了一条清晰的分界线。

    在东亚，中世纪的第三个阶段，是相对平稳的多政权并存阶段，辽、宋、夏、金、元等政权先后出现。需要指出的是，一开始，这些政权之间还能保证相对和平，边境也相对固定。不过，13世纪蒙古帝国的崛起，打破了这个看似“和平”的局势。蒙古人建立的政权对中原的金和南宋发动了灭国的战争，再次统一了中原。

    # 西方

    在中世纪，罗马帝国东边和西边的历史走向完全不同。

    在中世纪早期，东罗马帝国保持着强大的经济和军事力量，称得上是一个幅员辽阔的强大国家。这里顺便说一句，东罗马帝国也就是拜占庭帝国，以后为了方便起见，我们统一使用拜占庭这个称呼。

    而在原先罗马帝国的西部疆域，出现了大约两个世纪的战乱时代，那是欧洲最黑暗的时期。在这个过程中，一些蛮族，比如哥特人和法兰克人都建立了自己的政权，成为了欧洲地区的新主人；而另外一些蛮族，比如汪达尔人和匈人，或许曾经一度取得过军事上的成功，但最终逐渐被历史淘汰了

    在中世纪的中后期，位于亚欧大陆中部的阿拉伯帝国崛起，阿拉伯文明进入了它的黄金时代。

    而到了中世纪后期，蒙古帝国崛起，蒙古大军几乎席卷了整个欧亚大陆

    欧洲城市起源

    要说中世纪时期规模最大、持续时间最长的战争，有两个，一个是我们之前讲过的英法百年战争，另一个就是十字军东征。

    战争的结果通常不是发动战争的人可以控制的。比如，为了挽救拜占庭的十字军东征，最终葬送了千年帝国拜占庭；再比如，我们在专题中讲到的，为了争夺法国更多领土的英法百年战争，最终让英国丧失了在欧洲大陆的领土。总的来说，战争都会摧毁掉现存的旧秩序，导致一些新的社会阶层崛起。

    # 西亚

    到了中世纪，具有世界影响力的文明也只有两个——一个是东方以隋唐宋为代表的中华文明，另一个则是阿拉伯文明。

    为什么会这样讲呢？因为相比之下，西欧各国都只有地方性的影响力。哪怕是繁盛一时的拜占庭帝国，也只能说是对周边产生了影响力。法兰克王国倒称得上具有世界影响力，因为它奠定了后来西方文明的基础，但是它本身存在的时间太短。

    在中国的唐宋时代的史书中，阿拉伯一般被称为“大食”

    # 走出中世纪的原因

    • 欧亚大陆北方游牧民族摧毁了中世纪已有的、已经开始僵化的政治格局。 在此之前，无论是阿拉伯还是拜占庭，都出现了发展停滞的问题。在中世纪后期，真正有活力的是西欧文明，当时它已经开始了文艺复兴。但是，在军事和经济上，西欧依然比不上阿拉伯和拜占庭地区
    • 十字军东征让西欧觉醒。 虽然从军事上讲，十字军东征是彻头彻尾的失败，但是东征的十字军却在东方看到了更先进的文明，并且带回了东方的生活方式，刺激了西欧的文明发展。
    • 黑死病导致欧洲社会重新洗牌，人们观念改变，人性复苏。 当人们看到教会对黑死病无能为力，那些离上帝更近的主教和修士也大量死亡，他们对上帝的信仰开始动摇。特别是当大家感到生命非常脆弱、明天具有不确定性的时候，就开始产生“活在当下”的想法
    • 欧洲经济的恢复和文化的复苏。
    • 一个走出中世纪的深层次原因就是，欧洲贸易的发展。 商业的繁荣除了带来财富外，还带来了几个重大的变化：
      1. 西欧沿海的城市都有了自己的港口，有些还有了船队。这些船队后来变成了大航海时代的远洋舰队。
      2. 为了便于国际贸易，许多城市港口建立了国际贸易站，为外国商人提供临时住所和货栈。外来商人数量的增加，让各国在主要港口建立了自己的领事馆来保护国民的权利。
      3. 服务于商人的商店、钱庄、教堂和娱乐场所也如雨后春笋般涌现。这些服务已经带有了近代社会的特征。
        简而言之，人类走出中世纪的原因很多，但最重要的是文明本身的进步，或者说源于人类自身的发展动力。

    # 革命的年代

    工业革命

    经济学家许小年先生讲过这样一个历史观点，人类历史上只发生过一件事，就是现代化。在此之前属于前现代。而他所说的现代化的核心，其实就是工业化。因此，工业化之前发生的事，我们才称之为历史。而之后发生的事情，则可以说是我们当下所正在经历的。

    工业革命让英国成为了当时世界的经济中心，但是随着大量工业产品被生产出来，英国也面临着一个亟需解决的问题，这么多商品该卖给谁呢？为了解决工业商品的销售问题，英国开始改变它实行了两百多年的重商主义国策，转而积极推行自由贸易政策。它率先取消了贸易限制，通过开放自己的市场来换取国外市场
