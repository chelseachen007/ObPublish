    # 美洲

    # 美洲的历史

    我们以哥伦布到达的时间来了解美洲的历史

    # 哥伦布到达前

    美洲早期历史的记载很少，且缺少与外部世界接触的间接记载，所以很难讲

    在公元1500年之前，美洲主要的文明集中在中美洲和南美洲的西部，比较大的文明中心有三个——玛雅、特奥蒂瓦坎，以及印加。

    # 玛雅文明

    在哥伦布到达美洲时，玛雅人的文明水平基本处于新石器晚期和铜石并用的过渡期。

    但是玛雅人在数学、天文历法、艺术建筑方面有极高的成就，也有自己的文字，也有邦国这种复杂的社会组织，而且也有酋长、宗教祭祀、贵族和奴隶，也就是说社会已经分层。从这个角度来讲，说玛雅已经进入了文明状态也没什么大问题。

    玛雅文化可以追溯到公元前2000年左右，范围是从今天墨西哥的东部和南部，一直延伸到中美洲的中部，也就是今天的萨尔瓦多和洪都拉斯。

    到了公元前3世纪，玛雅人发明了自己的象形文字。玛雅人的书写系统极为复杂，很多“字”其实相当于其他语言中的一个短语，表达一个复杂的意思，因此这种文字非常难学，通常只有祭司阶层才掌握。

    此外，玛雅人还发明了20进制。这种计数方法极为复杂，如果想做加减乘除，难度比10进制高得多。

    2012世界末日的预言 是由于玛雅人采用20进制，于是他们的一个世纪是400年，非常长。每一个世纪，他们都认为是旧的时间的终结，新的时间的开始。

    玛雅文明的古典时期是从大约公元250年到900年。玛雅人在🇬🇹 危地马拉和🇲🇽 墨西哥建立了多个大型城市，而成规模的居民点更是数以百计。

    由于位于墨西哥中部的特奥蒂瓦坎文明此时也在崛起，并且发动了对玛雅城邦的战争，从9世纪开始，玛雅文明各个地区内忧外患，表现为整体性的政治崩溃，玛雅的古典文明就此终结。

    从1000年左右到1520年左右，是玛雅文明的后古典时期，他们在尤卡坦半岛建立了许多城市，并且留下了很多金字塔和神庙，特别是著名的卡斯蒂略金字塔。

    # 特奥蒂瓦坎文明

    特奥蒂瓦坎文明起始于公元前200年左右。

    由于缺乏文字记载，今天人们依然搞不清楚特奥蒂瓦坎人的起源和他们的生活。今天我们知道这个文明的存在是因为他们留下很多巨大的遗迹和周边文明提到过。
    有意思的是，在这个文明消失得无影无踪之后，特奥蒂瓦坎是后来的阿兹特克人对这座城市的称呼，并把他视作圣地。

    在大约公元600年，特奥蒂瓦坎文明达到顶峰。日月金字塔所在地是整个文明的中心，周围有大约10万到20万人居住。他们修建了很多巨大的砖石建筑，里面有精美的壁画。但是在650年至750年间，特奥蒂瓦坎文明突然消失了，城市被遗弃，人口不知所终。

    最终，来自北方的阿兹特克人于12世纪末入主墨西哥谷地，成为这里的新主人。阿兹特克人建立了很多城邦，并且和当地一些部落结盟，形成了阿兹特克帝国，成长为中部美洲势力范围最大的政权。不过阿兹特克帝国并不是一个中央集权的国家，更像是一个城邦联盟，各城邦向帝国进贡。

    1520年，西班牙殖民者就来到了这里。在1521年轻易地攻陷了帝国的首都特诺奇蒂特兰，俘获并处决了当时的皇帝。此后，墨西哥全境就成为了西班牙的殖民地。

    # 印加文明

    # 哥伦布到达后

    在大航海时代，先后有两批殖民者到达了美洲大陆。最早来到的是西班牙人和葡萄牙人，他们很快占据了今天墨西哥以南美洲的广袤领土，以及今天的美国南部。接着较晚到达美洲大陆的，是荷兰人、英国人和法国人，明显作为后来者，他们只能在西班牙人和葡萄牙人势力范围之外的北美洲中部和北部落脚

    不过，虽然这些欧洲人都是殖民者，但不同国家之间的具体目的却不一样。西班牙人和葡萄牙人来美洲的直接目的，都是为了赚钱。他们中的大多数人，并不是真的想世世代代生活在美洲。而荷兰人、英国人和法国人则不同，他们很多人都是为了逃避宗教迫害才来到美洲的，因此他们中不少人打算移居到新大陆后就不回去了。

    这两种不同的想法，导致后来北美殖民地和墨西哥以南的拉丁美洲殖民地，走上了两条完全不同的发展道路。

    # 拉丁美洲

    因为只以获得财富为目的，殖民者们很少会有发展和经营美洲的意愿，因为掠夺往往来得更快。

    但还是有部分想要经营城市的，但这光靠冒险家不够，后面殖民者政府又派了文官去治理。但这明显和冒险家的利益是冲突的，所以他们经常会爆发矛盾。这种殖民总督的独立性，对后来拉美的一些殖民地再独立出小国家影响很大。而不像北美的殖民地那样，后来形成了大的统一的国家。

    拉丁美洲的人口阶层

    在殖民早期的经济是采矿业，但是那只集中在少数有银矿的地区。而在其他广大的殖民地地区，最终占主导地位的是种植园经济。

    拉丁美洲那时候的种植园里，大多种植的是比粮食更值钱的经济作物，包括甘蔗、可可，以及后来的咖啡、香蕉、橡胶等。然后殖民者们再将种植园生产的产品出口到欧洲，换取欧洲的工业品。

    # 北美

    比较有代表性的是🇬🇧 英国占领的十三块殖民地，也是🇺🇸 美国的前身

    不少英国殖民者，还真是打算到了新大陆从此就不再回去了，或者换种说法，他们打算把英国的疆域扩展到海外。因为目标不同，所以西班牙人常常是少量的男性冒险家只身前往新大陆，而英国人大多则是拖家带口来到美洲。

    前提：在尝试殖民美洲之前，英国人就曾经有过殖民爱尔兰的经验。

    在开始的几十年里，英国人在北美建立殖民点的尝试都失败了。直到1607年，他们才在今天美国弗吉尼亚州的詹姆斯敦建立了第一个殖民点。

    英国商人经营殖民地的弗吉尼亚公司连哄带骗，动员了600英国人移民到新大陆。不过由于在海上遇到了风暴，最终到达詹姆斯敦的只有60人。不难想象，这么少的移民，不要说发展了，生存都成问题。
    直到后来英国人发现，在弗吉尼亚种烟草有利可图。于是他们开始改种烟草，并将烟草卖回到欧洲。就这样，殖民点生存了下来。

    后来弗吉尼亚公司动员了更多的英国人移民美洲，并给出了诱人的条件，参与移民的人都能在新大陆获得100英亩的土地。

    英国人几乎不和当地印第安人通婚，因为这些英国人在北美大多都有自己的家庭，这和西班牙殖民者是很不一样的。

    然后，他们向南发展到西印度群岛，那里气候温和很适合种甘蔗，但是英国人原先生活在北纬50度以上的地区，很难适应西印度群岛北纬20度左右的热带气候。

    于是，他们想到了从非洲购买奴隶来劳动。很快，西印度群岛上的欧洲人和黑奴的比例成了1:4。黑奴人数的增多，导致这里经常出现奴隶暴动，社会很不稳定。所以一些庄园主虽然赚到了钱，但是很快就失去了吸引力。

    英国人减少对西印度群岛的移民后，法国人来到了这里殖民，他们特别注重对岛国海地的经营。但是由于白人和黑人之间的矛盾无法调和，后来爆发了30年的革命，最终黑人赶走了殖民者，海地成为第一个由黑人领导的共和国。

    从1607年到1733年，英国在北美洲大西洋沿岸建立了一系列殖民地，其中有十三个在今天美国境内，它们也就是后来进行美国独立革命的十三州。

    # 三种殖民地

    • 由王室授予特殊经营权的殖民地。具体来说就是，英国国王会向殖民者们颁发一个特许状，给予这些殖民者在殖民地的经营权。比如：马萨诸塞殖民地
    • 英国国王赐给贵族的领地。贵族领地的所有权属于贵族，但是管理权属于当地人选举出来的议会。 比如：马里兰
    • 从一开始就自治的殖民地，也称为殖民者契约殖民地。它们是自由移民之间自己订立契约建立起来的。比如：普利茅斯殖民地
