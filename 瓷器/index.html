瓷器

最近编辑于 Dec 28, 2023

    瓷器虽然没有被列入中国的四大发明，它对文明的重要性，以及它对中国产生的商业价值绝不逊于四大发明中的任何一个。直到现在，在材料科学领域，陶瓷这种材质，依然发挥着重要的作用，我们今天很多日常的生活用品也都是陶瓷制品。

    最早的瓷器可能在东汉就出现了。但是，出现某项技术和能够将这项技术量产，这完全是两回事，中间需要经过漫长的发展和技术的沿革。

    虽然中国在东汉末年就已经具备了生产陶瓷的要素，但当时人们只能制造出少量的瓷器，并不能量产。烧制瓷器有三个关键要素——高岭土、上釉技术和高温，这三项技术基本上都是到了隋唐时代才被攻克的。

    # 量产的问题

    # 温度

    铁匠们用风箱把炉火烧得很旺，把铁都烧软了。这件事不难办到，因为打铁的炉子很小。但是，要想大批量生产瓷器，就需要修建大型瓷窑。而这时候，要想长时间把窑内的温度维持在1200度以上，就很困难了。

    为了解决这个问题，就需要给炉窑修建一个高高的烟囱，让烧热的空气上升，同时下面能吸入新的空气，这样就能给炉窑及时补充氧气。后来，中国南方的瓷器工匠们想出了一个更好的方法，他们利用南方多丘陵的地势，将瓷窑依山而建。这些瓷窑通常有几十米长，高处比低处能高出10米左右，起到了高高的烟囱的作用，这样能让瓷窑长期保持高温

    # 去杂质

    纯净的高岭土成分是二氧化硅，三氧化二铝和少量的结晶水。但是绝大部分高岭土并不纯净，其中都含有铁等杂质。这样烧制出来的瓷器颜色发黑发暗，既不美观也不结实。

    在唐代，窑工们能够控制高岭土中杂质的含量，让瓷器本身具有似青非青，似绿非绿的色彩。这让高岭土不纯所带来的缺憾变成了一种美感，非常特别。由于那里是古代越国所在地，因此生产的瓷器也被称为越窑瓷器。从颜色来分，越窑瓷器属于青瓷

    到唐代中晚期，中国北方邢州的工匠们已经能把高岭土中的铁，去除得比较干净了，烧制出来的瓷器就是纯白色的，被称为“邢窑”。不过，由于中华文化讲究一种空灵的美，因此知识阶层更青睐于青瓷

    # 名窑

    宋朝，汝、官、哥、钧、定五大名窑，其中，汝窑、哥窑、官窑和钧窑都是青瓷器，只有定窑是白瓷。

    # 汝窑

    因为产地在河南省的汝州而得名，它位列五大窑之首。盛产青瓷

    • 质量上乘而且稀有
    • 它漂亮而精致，符合文人的审美。汝窑瓷器呈天青色，这是一种深邃而透亮的淡蓝色

    汝窑瓷器烧制时间非常短，前后不过二十多年，而且产量很少，到南宋就已经一器难求了。今天，全世界的汝窑瓷器只有六十几件。

    # 龙泉窑

    到了南宋时期，由于北方发生战乱，中国的瓷器中心转到了南方，在浙江龙泉地区，出现了著名的龙泉窑。考古学家在那里发现了许多依山而建的窑址，最长的近百米，每个窑一次就可烧制两万多件瓷器。这么多的瓷器，不只是提供给中国人使用，很多要通过贸易输送到日本、东南亚、印度和阿拉伯地区。

    # 青花瓷

    出现在元朝,元朝统治者不同于汉民族的审美取向，以及它对于多种文明的包容造就了传世的元青花瓷器。

    蒙古人崇尚白颜色，蒙古族的很多服饰也是白色的。因此，他们对洁白如玉的白瓷比闪着青色光泽的青瓷更喜爱。

    从这里开始，出产白瓷的景德镇代替了出产青瓷的龙泉，成为中国的瓷器制造的中心和世界的瓷都。

    白瓷相比青瓷有一个巨大的优势，就是白色的瓷胎上容易绘制各种图案和上各种颜色。

    要想制造出大批高质量的瓷器，就需要能够养活大量的工匠，同时准备充足的燃料，这不是一个乱世能够做到的。因此，一个时代瓷器的制造水平，可以看成是当时经济和国力高低的晴雨表。事
