拜占庭帝国

最近编辑于 Dec 26, 2023

    拜占庭帝国，也就是东罗马帝国。

    拜占庭帝国，可以说是中古欧洲最长久的专制君主制国家。它的统治时期长达一千年，横跨了11个世纪，经历了九十多位君主。

    实际上，在这一千年间的时间里，拜占庭经历了十几个王朝，各王朝之间并没有血亲关系，只不过它的国祚基本是延续的而已。用一句俗话来解释就是，换人不换国。

    # 拜占庭的起源

    为什么三世纪之后，罗马帝国会逐渐放弃曾经非常繁荣的罗马城，转而将帝国的中心东移？

    第一个原因是，由于三世纪的动乱，罗马和它周边地区受到严重的破坏，帝国要在经济上支持这样一个大都市已经很困难了。

    第二个原因是，在历史上，罗马帝国的东方就是比西方更发达，也更加富有。因此，当罗马帝国开始走下坡路时，将统治中心东移几乎是一个必然的选择。而君士坦丁堡这个地方，地处海陆交通的要道，因此就被选定为新的首都。

    # 罗马和拜占庭的区别

    虽然拜占庭帝国直接继承了罗马帝国，但拜占庭文明和罗马文明其实有很多显著的区别

    # 拜占庭是政教合一的君主制国家。

    虽然罗马也有自己的国教，但是罗马的多神教对政治的干预并不多，它只是罗马人用于维持向心力的工具。但是拜占庭帝国所接受的基督教则不同，它是一个制度非常严密，而且影响人们生活的一神教。

    罗马的君主虽然也被成为“皇帝”，但国家总的政治制度其实更接近民主制度，皇帝的权力往往会受到制约。
    拜占庭的皇帝则不同，它更接近古代中国的皇帝。这个人既是整个帝国的象征，也是最高的政治领袖、军队的最高统帅、最高的司法裁判者和最高的宗教领袖。

    # 拜占庭是一个希腊化的国家。

    罗马在建国和发展起来之前，就将拉丁语变成了帝国的官方语言。但罗马帝国的东部，依然比较崇尚希腊文化。
    到了620年，拜占庭的希拉克略皇帝首次让希腊语取代拉丁语，成为帝国的官方语言。这使得拜占庭在文化上逐渐和罗马分家。

    # 罗马文明总的来讲是一个扩张的、输出的文明，而拜占庭文明是一个收缩的、输入的文明

    在古典文明时期，希腊和罗马都在不断向外输出它的文化，并且影响了周边民族。比如，后来在中世纪主导西欧发展的法兰克人，就接受了罗马文明。但是拜占庭在这方面则显得乏善可陈。不过，它吸收了基督教文化、佛教文化，疆域内部分人还皈依了伊斯兰教，因此，我们讲它是一种“输入”的文明

    # 几个阶段

    # 查士丁尼王朝

    查士丁尼可以说是中世纪早期欧洲最有作为的皇帝了，他一生都在为“再造帝国”这个理想奋斗，试图重新恢复罗马帝国的荣光。他通过武力收复了意大利故土和北非几乎所有的领土，特别是彻底打败了横行欧洲、非常野蛮的汪达尔人，让后者退出了历史舞台。但是，查士丁尼并不是一个穷兵黩武的皇帝，他虽然打败过东方的萨珊波斯帝国，却和对方签署了50年的和平条约，保证了东方的安宁。

    他主持整理了罗马法，编写了《查士丁尼法典》

    # 马其顿王朝

    在9世纪到11世纪中叶的时间里，马其顿王朝崛起，拜占庭帝国进入到了它的第二个辉煌时期。

    当时，拜占庭重新控制了巴尔干半岛，并在小亚细亚击败了阿拉伯人，甚至收复了意大利南部原属于西罗马帝国的领地。除了军事上的胜利之外，在这个繁荣时期里，马其顿王朝对文明产生重大影响的事情主要有三件：

    1. 在利奥六世时期，这位皇帝完成了对拜占庭法律的完整编纂。这部60卷的不朽著作成为后来所有拜占庭法律的基础，至今仍被许多学者研究。
    2. 在击败了新崛起的基辅罗斯之后，东欧的部落在拜占庭的影响下皈依了东正教，并且通过联姻和罗斯建立了关系。从此，基辅罗斯进入文明社会，逐渐演变为今天的乌克兰、俄罗斯和白俄罗斯。
    3. 大力发展贸易，将君士坦丁堡发展成为欧洲最大和最富有的城市，同时也是东西方贸易的纽带。

    另一方面，马其顿王朝之所以被认为是拜占庭由盛转衰的转折点，则是因为它破坏了拜占庭原来的土地制度，大量的自由农民土地被剥削，纷纷沦为农奴，这严重破坏了帝国的统治基础。从此之后，帝国对于底层民众完全失去了向心力，维持疆域只能靠武力了。而由于没有民众补充兵员，拜占庭帝国只能靠雇佣军作战。在强敌环绕的巴尔干和小亚细亚地区，拜占庭很快遭受到了一系列的军事失败，最后不得不求助于西方的基督教世界，从而引发了一场大战，这就是我们前面讲到的十字军东征。

    # 为什么能屹立千年

    # 地理位置

    拜占庭文明的中心是希腊半岛、巴尔干和小亚细亚，那个地方地处东西方交往，特别是商业往来的要道。当时无论是地中海、黑海周边的贸易，还是亚欧之间的贸易，都绕不开这里。

    希腊半岛、巴尔干地区以及安纳托利亚这片区域，在地理条件上还有一个特点，就是多山，易守难攻。即便到了第二次世界大战时期，强大的德军也没有能在那里讨到便宜。

    # 文化影响

    在古典文明走向终结的时候，西欧战乱不断，东亚也正值南北朝时期，可以讲当时这两个地方的文明几乎是被摧毁殆尽，但同一时期，拜占庭却继承了希腊和罗马的文化。这种优势让它成为了当时世界上文明程度最高的地区

    拜占庭人继承了古希腊人重视科学和教育的习俗，也和罗马人一样，非常重视少儿的教育。他们将算术、几何、音乐和天文视为“四艺”，并结合了哲学、修辞学、希腊语，一起制定了学童的必修科目。

    在高等教育方面，拜占庭在当时的世界也是处于领先地位的。博洛尼亚大学和君士坦丁堡帝国大学

    有了这样的教育作为基础，拜占庭无论是在和建筑相关的土木工程领域，还是在与军事相关的冶金术，甚至化学方面，都超越了之前的各个文明。

    建筑领域上有 圣索菲亚大教堂和 地下水宫殿

    军事技术上，拜占庭人发明了使用石油的火焰喷射器，它也被称为“希腊火”。

    在生活方面，拜占庭人继承了古罗马人日常沐浴的习惯。跟不爱洗澡的中世纪欧洲人相比，拜占庭人生活习惯要洁净得多。而且，拜占庭人不仅重视自己身体的清洁，还会定期对街道进行冲刷，以防止传染病的传播。君士坦丁堡在上千年的时间里，经历了多次瘟疫依然屹立不倒，这和它注重城市卫生是有很大关系的。

    # 经济优势

    君士坦丁堡特殊的地理位置让它成为中世纪欧洲最大的港口，世界各地的商船汇集于此。同时，它也是丝绸之路的一个终点。除了君士坦丁堡，亚历山大、安条克、萨洛尼卡和特拉布宗也是帝国的重要贸易港口。在长达上千年的时间里，帝国的主要收入都来自商业。除了征收的关税和贸易税，还有过境税、入城税、不动产转手税等名目的税收。

    手工业。公元6世纪之前，丝织业由古代中国人垄断，但是后来，蚕种和养蚕技术在查士丁尼一世时期传到了拜占庭。从此，丝织业就成了拜占庭最重要的手工业。拜占庭在希腊南部建立了国营的养蚕、丝织产业，严格控制丝绸生产、纺织和销售，严禁价格投机。为了获得更高的利润，拜占庭还不断改进纺织工艺。

    除了丝织业之外，拜占庭的冶金业、艺术品和珠宝制作也相当发达。

    # 衰落的原因

    # 军事衰落

    拜占庭到了中期，虽然商业非常繁荣，但是农民越来越贫穷，很多沦为农奴，帝国失去了士兵。到了帝国后期，陆军甚至只得聘请雇佣军，他们有些甚至来自拜占庭的主要敌国奥斯曼

    而拜占庭海军也同时衰落。由于海军无法靠雇佣兵作战，因此它只能依靠威尼斯和热那亚的舰队保卫海上通道。

    # 中央政权对地方的控制力不断减弱

    在帝国末期，所剩不多的几个残余省份几乎都成了独立王国，除了在名义上效忠拜占庭，不对中央政府承担任何义务。这些省份既不纳税，也不提供士兵，朝廷的政令几乎不出君士坦丁堡。帝国几乎没有收入，只能依靠变卖皇室财产、土地和借高利贷度日

    # 在政治制度、宗教和文化上都没有明显的进步。

    中世纪有五个主要的文明中心，从西往东分别是西欧、拜占庭、阿拉伯、印度和东亚。这里面，西欧和东亚都是在原来战乱的基础上通过民族融合诞生的新文明，阿拉伯文明更是从无到有产生的，因此它们都具有较强的活力。而拜占庭文明虽然是这五个文明里起点最高的，但它却是进步速度最慢的。虽然进行了很多改革，但是在政治制度、宗教和文化等各方面，都非常守旧

    # 但是

    对整个世界来说，拜占庭文明衰亡的影响也不都是负面的，甚至可以说，拜占庭的衰落和后来1453年君士坦丁堡的陷落，间接推动了西欧地区的文艺复兴和科学发展。

    因为在此期间，拜占庭流亡学者将古希腊的语法、文学研究，以及阿拉伯文明的数学和天文学知识，带到文艺复兴早期的意大利，他们还给西欧带去了大量的科学著作，包括植物学、医学和动物学，以及拜占庭经院学者对亚里士多德物理学的改进。
