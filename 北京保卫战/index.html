北京保卫战

最近编辑于 Feb 28, 2024

    # 备战

    八月十九日。于谦召开了他的第一次军事会议，必须说明的是，这位兵部侍郎虽然是个与军事打交道的主官，之前却从未指挥过军队。算是书生上阵

    于谦虽然是文官，但他对兵法也有研究，排兵布阵很有一套，相信是小时候看课外书打下的基础。

    他做了两件事

    1. 调部队
      • 备操军，包括两京备操军、河南备操军；2.备倭军，包括南京备倭军、山东备倭军；3.运粮军，包括江北所有运粮军；4.宁阳侯陈懋所部浙军（战斗力较强）。
    2. 解决粮食问题
      • 在离京城不远的通州，储存着很多的粮食，够京城的人吃一年。
      • 但大臣们却主张烧掉，因为运不进来，不仅需要安排民工运输还需要安排军队护卫。
      • 但有水平的人只需要一道命令就可以一石三雕，“所有受召军队进发时应由通州入京，士卒各自取粮，并运送至京城。” 调兵、粮食安全、运输人工

    在于谦的努力和调配下，到九月初，各路人马纷纷赶到，京城的兵力达到了二十二万，且粮食充足，人心也逐渐稳定下来。

    中间还发生了文官殴打太监，这件事后所有的人都对他报以敬佩的目光。

    随后就是最后个麻烦事，另立皇帝。

    朱祁镇三岁的儿子朱见深肯定不行，那只能是朱祁钰了。

    但他拒绝了，不是我们推红包一样的虚情假意。是真的害怕了，想要活着。

    可是事情已经不是他能控制的了。他做也得做不做也得做！

    # 开战

    # 战前部署

    也先带着朱祁镇，执行他的”撞门“计划，想着一声令下就打开入京的大门宣府和大同，却没有一个地方接纳。

    既然”撞门“行不通，那就开战！

    也先并不是傻瓜，他没有带领军队去攻击宣府和大同，那两位打不过还惹不起吗，他选择从紫荆关寻找突破。

    也先的军队十分强悍，骑兵以猛虎下山之势直扑紫荆关，在喜宁的引导下（所以说叛徒最为可恨），也先仅用了两天时间就攻破了这座关口，守备都御史孙祥战死。

    当年朱文正几万人面对陈友谅六十万大军能守下洪都，二十二万守下北京看着也不是难事。

    但北京外城就有九门，平均一下每个门只有两万守军，而且当时的北京虽然没现在的大，但也是相当大的，很难做到互相支援。

    所以明军虽然在总的人数上占优，但平均到每个门的防守却是不折不扣的劣势。更何况兵力素质也相差很多。

    石亨认为，要想退敌，最好的方法就是坚壁清野，等待敌军疲惫，自然就会退军了。

    但只有于谦反对，说出自己的观点：

    “也先率大军前来，气焰已经十分嚣张，如果坚守不出，只会长他们的气焰，我大明开国至今已近百年，昔日高皇帝布衣出身，尚可纵横天下，横扫暴元，我辈岂惧小小瓦剌！”

    做出了如下部署

    1. “大军全部开出九门之外，列阵迎敌！”
    2. “锦衣卫巡查城内，但凡查到有盔甲军士不出城作战者，格杀勿论！”
    3. 九门为京城门户，现分派诸将守护，如有丢失者，立斩！
      • 德胜门，于谦！
    4. ”临阵，将不顾军先退者，立斩！临阵，军不顾将先退者，后队斩前队！“
    5. “大军开战之日，众将率军出城之后，立即关闭九门，有敢擅自放入城者立斩！”

    # 激战

    # 先攻西直门

    也先决定打西直门，上千名瓦剌士兵挟持着俘获的百姓向西直门发动了试探性的进攻。

    西直门的守将是刘聚，他迅速作出了反应，派遣部将高礼、毛福寿迎敌。在土木堡之战中，他们很多人都失去了自己的战友甚至亲属，满腔怒火正无处宣泄，现在这些杀戮自己同胞的仇人竟然还敢找上门来，真正是岂有此理！此仇不报，更待何时！

    瓦剌兵惊呆了，他们以为是个美差，是让他们来接受投降，先进城掠夺一波。
    当也先看到逃回来狼狈不堪的瓦剌士兵时，他已经明白，眼前的敌人不是牛羊，而是虎狼。

    这时，卖国贼喜宁又出现了，提出了把朱祁镇放在显眼的位置，并公开通知对方前来迎接的计策，如果对方来接，就可以谈条件，索要钱财和利益，如果不来的话，明朝就会理亏，从礼法上讲也是一件丢人的事情。

    # 尝试谈判

    于谦解决的办法是升了两个六七品的小官至正四品，派他们去谈判。这不是谈判，这是调侃，是侮辱。
    更可笑的是，也先对于明朝的官制和人员并不清楚，他还一本正经地要和对方谈判。也先不懂，但是卖国贼懂，他让也先拒绝和他们谈，并表示他们的谈判对象仅限以下四人：于谦、石亨、胡濙、王直。

    于谦回复：“我只知道手上有军队，其他的事情不知道（今日只知有军旅，他非所敢闻）！”

    # 德胜门受挫

    也先知道德胜门已有准备，就先派出一波斥候，还没有到德胜门就发现了明朝骑兵，而且神色慌乱，装备不整，他们跟踪追击，发现一路都是这种情况。回报判断，德胜门还没做好准备。

    瓦剌派出了一万大军进攻德胜门，带队的主将是也先的弟弟孛罗，这支军队是也先的精锐，他派出主力作战，表明其志在必得的决心。

    当瓦剌骑兵冲入这片空旷的民居时，突然从前方两翼冲出大队士兵，堵住了瓦剌前进的道路。与此同时，大队士兵在瓦剌军后面出现，切断了他们的退路。而道路两边埋伏着神机营复仇的火枪，他们将用手中的火枪痛击这些入侵者，为之前死去的战友复仇，并赢回这支精锐部队的荣誉。

    # 总攻定安门

    也先失去了他的耐性，他下达了总动员令，命令所有骑兵对京城九门同时发动总攻。
    这时他还能撤走，但是就算走，也要赢一把再走！这就是死于要面子

    他经过仔细考虑，为自己这次表演选定了目标——安定门。也先带领着他的精锐主力向安定门扑去，但他比他的弟弟要谨慎得多，一路上他都小心翼翼，唯恐中埋伏。但让他吃惊的是，一直到安定门前，都没有遇到过任何麻烦，也没有任何伏击者出现，这更让他确定了安定门是京城防守的弱点所在。

    但这里守着的是和也先有死仇的石亨，他们本来互有胜负，但是在后来的阳和之战中，石亨输掉了他所有的一切。

    石亨仔细研究了瓦剌骑兵的特点，他利用这仅有的一个月时间加紧训练手下的士兵，教导他们作战方法和战术。

    但事情的发展超出了石亨的预想，石彪竟然如此威猛，仅凭一己之力就击退了也先，这样也好，通常打落水狗总是容易的。

    # 镗力战不支，欲入城

    #勇气
    随后也先军发现西直门易攻，将主战场转移到西直门。西直门的守将是孙镗，他是一个比较有能力的将领，他带队在门前迎战，率领守军主动冲击瓦剌军前锋。他本人武艺高强，勇猛异常，身先士卒，手持大刀亲自参加白刃战，斩杀多名瓦剌士兵（斩其前锋数人）。

    但人越打越多，孙镗产生退意，只好退到城门前对着城头喊话：“我已支持不住，放我军入城！”

    城头程信在城头看得一清二楚，也明白孙镗也并非贪生怕死，实在是支持不下去了，可是他军令在身，而且他也是一个比较死板的人，拒不开门，并说：我理解你的辛苦，但是我有军令不能开门，其实你只要打退了敌人，就可以进来了，我在上面会为你加油的。

    孙镗明白了进城的只能是胜利者或者尸体，拨转马头，转向了激战正酣的战场。反正也进不去了，就战死在这里吧！也先，老子跟你拼了！士兵们被他的勇气所鼓舞，无不奋力死战，明军士气大振，稳定住了局面。

    正在战局相持不下之际，石亨终于赶到。

    # 败退

    也先最后决定他利用手中的朱祁镇，想同城内的人谈判，其实他的要求也不过分，给点钱财让他有个台阶下，也就够了，可城内的于谦根本就不搭理他，于是他就武力进攻，可总是被打得落花流水，落荒而逃。

    于是他决定转而求其次，攻击居庸关，这样进可攻，退可守，进退自如。

    但天气不站在他这边，那年的雪比以往来的更早一些，守城的罗通效仿朱文正浇水打造了一个冰城。

    也先在居庸关外痴痴地等待了七天，希望眼前的这座冰山能够融化，可他的得到的只是城内射出的弓箭和火铳，以及守军无情的嘲笑。

    似乎是为了怀念自己那最终未能实现的梦想，也先在离北京城外不远的地方扎营，度过了在京城的最后一个夜晚。

    但是于谦为他准备了一个大礼，那就是大炮。
    为什么之前不用？因为白天有误伤朱祁镇的可能，影响实在不好，舆论压力太大。晚上黑灯瞎火的时候就算一不小心“误伤”或是“误杀”了太上皇阁下，那也是可以理解的。

    也先同志带着他还没有做完的美梦，连夜离开了这片伤心之地。
