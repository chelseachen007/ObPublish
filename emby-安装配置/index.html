    # emby 安装配置

    # 一、Emby的系统要求

    在安装Emby之前，我们需要确保您的设备满足以下系统要求：

    1. 操作系统：Emby支持多种操作系统，包括Windows、macOS、Linux等。请确保您的设备运行了一个兼容的操作系统版本。
    2. 处理器：建议使用具有较高性能的处理器，以确保流畅的视频转码和播放体验。
    3. 内存：至少2GB RAM，建议4GB RAM或更高。
    4. 存储空间：至少20GB可用存储空间，以存放Emby的系统文件和媒体内容。

    # 二、在不同操作系统上安装Emby的步骤

    1. 在Windows上安装Emby：

    a. 访问Emby官方网站（ https://emby.media/download）并下载适用于Windows的安装包。

    b. 双击下载的安装包，按照提示完成安装过程。

    c. 安装完成后，运行EMBY Server，然后在浏览器中访问 http://localhost:8096 进行初次配置。

    2.在macOS上安装Emby：

    a. 访问Emby官方网站（ https://emby.media/download）并下载适用于macOS的安装包。

    b. 双击下载的安装包，将Emby Server应用拖放到“应用程序”文件夹。

    c. 运行EMBY Server，然后在浏览器中访问 http://localhost:8096 进行初次配置。

    3.在Linux上安装Emby（以Ubuntu为例）：

    a. 首先SSH连接实例，获得超级权限：

    _sudo -i_

    b. 下载emby：

    _wget <https://github.com/MediaBrowser/Emby.Releases/releases/download/4.7.11.0/emby-server-deb_4.7.11.0_amd64.deb>_

    c.更新软件包列表：

    _sudo apt-get update_

    d. 安装emby Server：

    _dpkg -i emby-server-deb_4.7.11.0_amd64.deb_

    e.安装完成后，Emby Server将自动运行。在浏览器中访问 http://localhost:8096 进行初次配置。

    # 三、安装后的初次配置

    在完成Emby的安装后，我们需要进行一些初次配置，以便让Emby更好地为您服务。以下是配置的主要步骤：

    1. 使用浏览器访问Emby的Web界面：根据您的设备和网络环境，在浏览器中输入相应的地址（通常是http://localhost:8096）。
    2. 选择语言：在界面中选择您喜欢的语言。Emby支持多种语言，包括简体中文、繁体中文、英文等。
    3. 创建管理员账户：输入用户名和密码，创建一个管理员账户。作为管理员，您将能够管理Emby服务器的所有设置和内容。
    4. 配置媒体库：按照提示添加您的媒体文件夹。您可以选择电影、电视剧、音乐等不同类型的媒体文件夹，Emby将自动抓取相应的元数据。
    5. 配置远程访问：如果您希望在家庭局域网之外访问Emby服务器，可以在此步骤中配置远程访问。只需启用“允许远程访问”选项，并确保您的路由器已正确配置端口转发。
    6. 完成配置：在完成以上步骤后，点击“完成”按钮。Emby将为您准备好媒体库，让您尽快开始观影之旅
    7. 把媒体库的元数据下载器选择为TheMovieDb
