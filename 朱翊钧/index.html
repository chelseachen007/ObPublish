    # 抗倭援朝

    万历十九年(1591)八月，福建巡抚赵参鲁奏报：
    根据琉球使节反映，近日突然出现上百来历不明者，前往琉球朝鲜一带收购海图以及船只草图，并大量收购木材火药，用途不明。

    🇯🇵 日本的头头丰臣秀吉在北九州的荒野上修了一座城池。当这座城池建好的时候，站在城楼的最高点，就可以清晰地看到一个地点——朝鲜海峡。

    万历二十年五月二十四日，倭军仅两个月拿下朝鲜八道中的七道，陈兵鸭绿江边。

    但剩下的一条道有一个狠人，叫李舜臣，接连在水上挫败日军，但陆地上该丢还是丢了。

    万历虽然不上朝，但是管事，在大臣在是否出兵的上吵的不可开交的时候，他一锤定音：“应该早日出兵救援（宜速救援）！”

    第一战，祖承训带领上千人进入平壤，被小西行长埋伏，死伤两千余人，几近全军覆没，副将史儒战死。

    万历随后派出兵部右侍郎宋应昌，准备正式开战，但宋应昌说：还需要等一个人两个月

    那只能找人外交了，沈惟敬。还真给他忽悠到了，拖了两个月。

    随后李如松带着辽东铁骑来了

    # 第二次援朝战争

    麻贵带领，

    老实人邓子龙，“此船即我所守之土，誓死不退！”

    万历二十六年(1598)十一月十九日中午，历时一天半的露梁海大战正式结束，日军精锐第五军全军覆没，史称“露梁海大捷”。

    在进行这场战争的同时，明朝还调兵十余万，围剿四川方向的杨应龙叛乱。在万历同志看来，这位叫杨应龙的土财主（土司），比丰臣秀吉的威胁更大。
