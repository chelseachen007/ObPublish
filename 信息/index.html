    # 信息

    比尔·盖茨在 Ted 演讲上讲了一个非洲人民抗疟疾的故事。
    这是一句完整的表达。
    但它是信息，不是知识。
    信息是对各种数据或文字的连接，它能准确表达一个意思，从而降低信息传递中的不确定性。

    # 什么样的信息最有价值

    一种信息，可以帮你从新的角度来核实现有信息的真实性，及早识破虚假夸大的信息；
    另外一种信息，是给你对于能源/财富的寻觅，提供一条全新的替代的通路。

    提高自身全方位多角度的获取信息的能力，是最好的投资，比物理世界很多投资要高效得多。一块钱换来的有效信息，让你识别真相，增加选择，可能避免犯下一百块钱甚至一千块， 一万块的错误。即使 99%的信息最后无用，只要 1%有用，就值了。

    低成本获取高价值的信息，以创造更多选择权 ；有更多选择权，就更有时间和资源去建设自身的信息获取和整合的能力，从而增加更多选择 ； 有了更多选择，才发现可以不做那些原来以为必须要做的事；看似无所事事，但背后决策机制，常人并不知晓。

    # 关于信息的分类

    我们往往会把战争新闻视为「严肃内容」，而会把明星八卦视为「娱乐内容」。但其实从某种角度来看，他们都是一回事。因为都是发生在遥远的地方，对我们当下的生活没有任何具体的影响。

    所以从是否引发行动的角度来看，它们都是娱乐类信息。

    而什么信息能引发我们具体的行动呢？主要是教程类、实用类信息。

    • 教程信息是指那些你能跟着步骤实现的信息。它们不能被夸大，也不能被歪曲任何东西，因为你可以跟着教程一步一步来复现每个步骤。而这个过程也意味着这些信息被你理解掌握，知道如何应用，才能转化为知识。

    • 除了教程，诸如如天气预报、股票价格等信息也很重要。因为这些信息能为你决策指导，引发你具体的行动。比如股票跌了是否需要继续买入还是卖出等。

    娱乐类信息就像是饼干，偶尔吃一块饼干是没问题的，但是如果每天都是以饼干为主，就会引发思想肥胖。

    注：可以用这个视角来审视自己的信息源，会发现许多信息源的内容都无法引发自己具体的行动。而后者是否很枯燥？取决于你是否有自己明确的目的。比如打游戏时候查看攻略就不会觉得很痛苦，反而会津津有味。所以如果觉得后者非常枯燥，不知道从何看起，大概率是自己缺少明确目的 —— 这在下文也有提及。
