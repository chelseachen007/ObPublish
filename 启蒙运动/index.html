启蒙运动

最近编辑于 Jan 5, 2024

    苏格兰启蒙运动和法国启蒙运动一样，也是一场伟大的思想解放运动，法国的启蒙运动奠定了现代国家的理论基础，那么苏格兰启蒙运动则是奠定了我们今天公民社会、商业社会和技术社会的思想基础

    # 法国启蒙运动

    # 苏格兰启蒙运动

    在伊丽莎白一世去世后，英格兰和苏格兰拥有共同的国王，但是它们却有各自不同的议会，是两个相互独立的主权国家。

    在光荣革命之后，英格兰和苏格兰的王室也不相同了。当时苏格兰是欧洲的一个弱国，不过当西欧各国从大航海和全球贸易中赚取了丰厚利润之后，苏格兰也参与了大航海。但是由于国力较弱，它很难在各个强国的垄断下分一杯羹。苏格兰在中美洲好不容易建立起一个不大的殖民地，很快却被西班牙给消灭了。这件事让苏格兰在经济上濒临破产。

    于是苏格兰王国决定，干脆和它的邻国，蒸蒸日上的英格兰合并。

    合并之后，🏴󠁧󠁢󠁳󠁣󠁴󠁿 苏格兰不再需要为国防发愁，苏格兰人就此赢得了一个和平的发展环境。

    这时很多苏格兰人就在思考，要如何发挥自己的优势，在未来世界中重新确立自己的地位。

    🏴󠁧󠁢󠁳󠁣󠁴󠁿 苏格兰也不行，在这样大背景下，苏格兰的知识精英开始启动了全民的启蒙运动。

    # 思想领域

    如果说法国启蒙运动在思想上的贡献主要表现为对于现代社会的构想，苏格兰启蒙运动的贡献则在于对人与社会的关系，人与人的关系的研究上。

    而当封建秩序被推翻后，在新的社会中，人与人的关系应该是怎样的呢？其实没有人考虑过。

    • 苏格兰启蒙运动中最早的思想家哈奇森认为，仁慈和道德感是人生与俱来的，就如同视觉和听觉一样。基于这种认识，哈奇森进一步指出，人们身上功利主义的原则让人通过做善事获得成就感和快乐
    • 休谟认为人性的善恶不是天生的，而是由我们过往的经验造就。因此人是会不断改变的，就像今天的我们和五年前的我们相比，其实已经有了很大的不同。
    • 亚当·斯密作为哈奇森的学生自然受到了后者的影响，但是另一方面他也接受了休谟的很多想法。

    # 催生了工业革命

    当时苏格兰的中心爱丁堡，科学家和发明家辈出，学术氛围浓厚，被称作“北方的雅典”。

    当时苏格兰的学者们坚信技术能产生工业的革命，工业和商业将会使人们更为自由，自由将带来文明与优雅，然后推动人类的进步。

    在这样的社会氛围下，科学和技术得到了结合，大大加快了技术革新的成功率和效率。并且英国当时十分重视对专利权的保护，这也极大地激发了人们进行科学创造的热情。

    # 开启了全球化的商业文明

    其中最主要的就是亚当·斯密提出的“自由贸易”政策，对资本主义的发展起到了巨大的推动作用。

    # 两次启蒙运动的异同

    # 目的不同

    法国启蒙运动的目的是反对王权专制和教会的权力，而苏格兰启蒙运动则是为了提高自己在联合王国中的地位。

    由于目标的不同，法国和苏格兰启蒙运动在具体做法上也并不相同。

    法国启蒙运动对欧洲封建制度和教会的批判是尖锐的。比如我们前面讲过，中世纪之所以过去在欧洲的名声很差，主要就是拜法国启蒙思想家们所赐。

    在苏格兰这边，苏格兰启蒙思想家们强调尊重社会既有的风俗和习惯。他们支持自由，却不反对君主和贵族。更关键的是，虽然这两国的启蒙思想家都认同人和社会的契约关系，但苏格兰启蒙思想家强调社会要有法律和制度，凡事要在制度框架内进行，而没有像法国大革命时期的革命家们那样鼓吹革命。

    # 参与的人不同

    法国启蒙运动的领导者们主要是具有先进思想的社会上层人士。但是另一方面，法国底层民众其实没有直接参与到启蒙运动当中，这也导致了他们其实并没有透彻地理解启蒙思想。

    苏格兰启蒙思想家们共同的特点是每个人都学富五车，但在个人经历方面并没有什么能让历史学家们大书特书的事件。他们甚至没有提出对于未来社会完整的构想，只是要通过改良社会制度建立一个更公平、更好的社会。苏格兰启蒙运动推崇的是渐进式改革，更尊重英国的传统和长期以来的治国经验。

    # 带来的影响不同

    法国社会的改变是经历了一番血雨腥风的。甚至到最后法国大革命开始失控，充斥着暴力行为。此后，法国又有几次不得不回归传统，然后再次爆发新的革命。

    相比之下，苏格兰启蒙运动则非常温和。虽然它采用了一系列尊重传统、根据经验、和风细雨式的改良。由于有全社会的参与，苏格兰启蒙思想可以影响到英国上层的立法者和行政官员，从而慢慢改变英国的国策，再慢慢改进英国社会。

    在这个过程中，我们看不到太多波澜壮阔的历史场景，这可能也是历史学家们很少花笔墨描写苏格兰启蒙运动的一个原因。不过，专业的政治学者很少会低估苏格兰启蒙运动的地位。
