阿拉伯

最近编辑于 Dec 27, 2023

    一提到阿拉伯文明，人们自然会想到伊斯兰教，想到穆罕默德。这主要是因为，在历史上大部分时间里，阿拉伯半岛都只有零散的部落，而没有统一政权，直到穆罕默德创立伊斯兰教，阿拉伯文明才迅速崛起。

    对于穆罕默德之前的阿拉伯的历史，今天绝大部分历史书都很少提及，主要是那个地区之前的文字记载有限。

    在公元一世纪之后，阿拉伯西北部就处于罗马帝国的控制之下，而其他地区则由帕提亚帝国和后来的萨珊帝国控制。从阿拉伯半岛上有文明出现，到公元610年伊斯兰教出现之前，都被称为阿拉伯的前伊斯兰时期。

    在阿拉伯半岛，只有南部很少的土地能满足农耕所需的用水。其他地区，不要说是农耕，连定居都很困难。这样的地理和气候条件，塑造了他们特殊的社会和文化。

    • 阿拉伯人通过部落的形式组织起来，各个部落不断赶着牛羊寻找有水草的地方，生活一段时间后，继续寻找下一个目的地，因此他们一直处于游牧状态
    • 生存环境的恶劣对生活在这里的人们还产生了一个影响，那就是大家必须共同保护部落财产，否则就可能面临死亡的威胁。
    • 贫瘠的生存环境对阿拉伯人的第三个影响，就是他们很难通过农耕生产填饱肚子，反而是因为地处欧亚非交界之处，很多阿拉伯人是以长途贸易为生的

    公元632年，穆罕默德去世，人们通常把这一年看作阿拉伯帝国的起始元年

    # 阿拉伯帝国的发展阶段

    # 四大哈里发时期

    公元632年，穆罕默德去世，但没有指定继承人，于是各部落停止进贡，开始自行其是。
    但是伊斯兰教的信仰已经在人们心里扎根，为了让各派穆斯林能够团结在一起，最后经过各派协商，决定由穆罕默德最信任的好友、他四大弟子之一的阿布·伯克尔出任穆斯林的领袖 。
    伯克尔继任后被授予“真主使者继承人”的头衔，在阿拉伯语中被称为“哈里发”。这里的“真主使者”指的就是穆罕默德。

    不过，伯克尔只担任了两年的哈里发就去世了。随后，穆罕默德其他三个弟子也接连被选举为哈里发。由于他们四个的年龄都比较大，所以这四任哈里发统治时间不长，加起来也只有29年。历史上，人们通常将这段时间称为“四大哈里发时期”。

    虽然持续的时间短，但值得一提的是，这四任哈里发都战功彪炳

    # 统一王朝时期

    在这一阶段，阿拉伯帝国正式确立了自己在世界文明的地位，无论是帝国的版图还是文明成就都到达了一个高峰。

    公元661年，第四任哈里发阿里在进行礼拜时被另一个教派的刺客刺杀。随后，阿里的儿子哈桑在一些穆斯林的拥戴下就任第五任哈里发。但不久，掌握最多军队，并且控制着埃及和小亚细亚的叙利亚总督穆阿维叶一世，威逼哈桑退位，并自立为哈里发，定都大马士革。这就是倭马亚王朝。

    倭马亚王朝后期的统治非常残暴，导致众多被征服民族的怨恨日益加剧。同时，被打压的什叶派和掌握政权的逊尼派的争斗也日趋激烈。终于在750年，也有说法是752年，阿布·阿拔斯-萨法赫联合各方力量，推翻倭马亚王朝的统治，建立阿拔斯王朝。由于阿拔斯王朝的旗帜多为黑色，因此中国史书称它为“黑衣大食”

    阿拔斯王朝统治期间，阿拉伯和当时的唐朝进行了一场规模较大的战争——怛逻斯之战。这是阿拉伯和中国进行的第三次直接冲突，也是规模最大的一次战争。

    阿拔斯王朝实际控制的疆域和人口，比不上之前的倭马亚王朝。不过，阿拔斯王朝在文明水平上达到了一个空前的高度

    # 多政权并立的时期

    从9世纪后半叶起，阿拉伯帝国开始衰落，阿拉伯世界分裂成了大大小小十来个国家。虽然我们有时还称呼它们为阿拉伯帝国，但其实它们已经属于不同的政权了。

    值得一提的有两个，法蒂玛王朝以及塞尔柱帝国

    # 政治结构

    “哈里发”的意思是“穆罕默德的继承人”，通常是伊斯兰政权政教合一的领袖，因此只能有一个人。后来，阿拉伯世界分裂，哈里发可能只是宗教领袖，而没有世俗的权力，成为军事强人的傀儡。
    而“苏丹”是一般意义上“首领”的意思，通常不是宗教领袖，因此苏丹可以有多个。

    在地方上，阿拉伯帝国仿照波斯帝国和罗马帝国的做法，建立了行政区（相当于行省），任命总督全权负责军事事务。但是和波斯帝国、罗马帝国不同的是，阿拉伯帝国的总督通常不管理税收，甚至不管非穆斯林的法律事务。

    非穆斯林只要按照规定交了税，自己还过自己的生活，帝国的官员并不过多过问。

    # 军事能力

    阿拉伯帝国在早期是非常善战的，并且在对外战争中不断获胜，很快就将领土从阿拉伯半岛扩张到欧亚非大陆，扩张的速度仅次于后来的蒙古帝国。而阿拉伯帝国在对外战争中能够不断获胜，原因主要有三个：

    • 首先是因为穆斯林们对伊斯兰教的信仰，使得阿拉伯帝国内部的凝聚力非常强。且周边罗马帝国和波斯帝国正在衰落

    • 其次，阿拉伯帝国的军人们有着自己独特的沙漠作战经验，特别是阿拉伯人非常擅于利用骆驼长途作战。

    • 最后，则是因为阿拉伯当时对其他宗教采取了比较宽容的政策。

      举个例子，第二任哈里发欧麦尔在占领耶路撒冷之后，没有迫害异教徒，对于基督教徒和犹太人，允许对方保留自己的宗教，但是要纳税。而对于皈依伊斯兰教的人，可以获得免税的优待。

    阿拉伯人在当时是最文明的。

    # 继承制度

    阿拉伯帝国的继承制度非常特别，它既不同于古代中国的嫡长子继承制，也不同于草原部落的兄终弟及制，也跟后来蒙古等部落的幼子继承制不同，而是一种被伊斯兰世界普遍接受的长者继承制，也就是由皇族中活着的、最年长的人来继承。

    对王位的野蛮争夺，使得帝国继承人往往并不是能力最强的，也并非资历最老的，久而久之，帝国难免陷入一种长期的衰落之中。

    # 农业

    阿拉伯帝国非常重视农业，统治期间农业技术取得了很大的进步，各种农业经验还被总结成书，在伊斯兰世界流传。通过灌溉、施肥、轮作和混种等技术，帝国的农业收成有了大幅提升，保证了城市的发展。

    古典文明之后又一次亚非欧之间的物种大交换，甚至现在你在中国能够看到的很多物种就是那个时期引进过来的。

    阿拉伯地区各种农作物如胡桃(核桃）、胡荽、胡豆（蚕豆）、胡麻、胡瓜（黄瓜）、胡蒜（大蒜）、胡萝卜、芹菜、波菜等种种蔬菜， 椰枣树、刺桐 、茉莉花、曼陀罗花等多种植物的引进与种植，影响了中国的农业生产和中国人的饮食结构。阿拉伯手工艺品如香料、药材、珠宝、乐器传入中国，丰富了中国人的社会生活。伊斯兰教的传入，给中国这片古老土地上带来一种新的文化价值观，直至后来影响了中国 10 个信仰 伊斯兰教少数民族的形成与发展。
    —-《唐宋时期中阿交往及其历史意义与当代价值》

    # 科学

    花拉子密

    伊斯兰世界也热衷于炼金术。不过，中国道士们炼丹对科学的发展并没有什么帮助，但是阿拉伯人和波斯人的炼金术对后来化学的诞生产生了巨大的作用。阿拉伯人发明和改良了许多实验设备和实验方法，比如蒸馏设备，蒸馏、升华、过滤和结晶等方法都是他们发明的。此外，阿拉伯科学家们还成功地提炼出纯酒精、苏打、硝酸、硫酸、盐酸、硝酸银和硝酸钾等多种化学物质。

    在医学和物理学领域，阿拉伯科学家也做出了巨大的贡献。在医学上，今天我们把医学分为外科、内科、骨科、眼科、神经科、妇科等，而早期对医学这么分类的正是阿拉伯人。他们在医学的很多领域，都曾经领先于欧洲人上百年。

    此外，他们在光学、物理学上的成就也很高。

    # 为什么？

    • 一个很重要的原因是，那里地处欧亚非交界之地，从欧亚非三个地区获取知识和信息都比较便利。
    • 在阿拉伯帝国时期，伊斯兰教还是比较宽容的，并且提倡兼收并蓄。 正是在这样的环境中，在欧洲停滞了的科学在阿拉伯帝国得到了发展。
    • 阿拉伯学者在治学方法上也比先前的文明有所进步。 他们非常注重基于量化实验的科学研究，修正了过去很多根据主观经验得到的不准确的科学结论。同时，阿拉伯也是世界上最早对学者们进行同行评议，并制定专业人士操作规范的地区。

    阿拉伯人在科技上的成就，为中世纪末期的欧洲人提供了丰富的理论、资料和研究方法，有力推动了全世界科技的进步和欧洲科学的复兴。文艺复兴和工业革命

    # 贸易

    从公元8世纪到12世纪的丝绸之路上，唱主角的就是阿拉伯人。虽然当时中国也积极参与其中，但并不是主导者，而更像是一个被动的参与者。

    古代中国和阿拉伯帝国的不同姿态，跟两地的气候条件和经济特点有关。

    在阿拉伯地区，从事农业生产是一个相当费力但是收益不高的经济活动，因为那里炎热干旱，非常不利于农作物的生长。不过，那里地处东西方交流的中心，对于当地人来讲，最方便、收益最高的劳动，反而是做生意。当然，在古代社会做生意要冒着生命危险，并非每一次长途贸易都能活着回来。因此，对于中原地区的农耕民族来讲，在土地上种庄稼是一个既无生命危险，又有稳定收成的事情，虽然一年忙到头只能勉强维生，但总比外出丢掉性命来得好。

    这样两种截然不同的思维方式，恰好形成互补。阿拉伯商人们来到中国，看到中国的农民和工匠们一天忙到晚，只有一点点收入，觉得自己赚到了便宜。中国人看到对方不远万里，历尽千辛万苦来中国做生意，自己在家门口就可以把东西卖出去，也觉得自己很合算。正是抱着这种各取所需的想法，才让贸易得以进行。

    # 文学

    • 《天方夜谭》（《一千零一夜》）
    • 阿拉伯人还热衷于整理、翻译和改编其他文明的古典著作。阿拉伯当时的首都巴格达，因为配备有一大批专门的翻译人才，被誉为“智慧之城”。

    # 哲学

    • 对西方世界影响最大、最深刻的，当属12世纪的伊本·鲁世德。他间接开启了近代欧洲的理性主义，这是阿拉伯文明对世界最大的贡献之一。
