    # 唐朝

    # 为什么唐朝会在军事、政治和经济都最强盛的时候，迅速衰落下去，并且一蹶不振？

    很多人把唐帝国的衰落归结于唐玄宗在后期懒政，贪图享乐，任用奸臣等等。但我想说，这些其实只是表面现象，唐王朝的危机在它不断扩张时就已经埋下了。

    # 这个盛世是被什么因素成就的？

    隋唐宋时期的经济，并不是农业、商业这种单一领域的崛起，而是一次系统性的繁荣。而这中间穿针引线，将农业、手工业、商业连成一体，共同组建起一个良性经济系统的，正是隋炀帝督办的大运河。

    隋朝时期，由于黄河流域，特别是关中地区，在两千年的开垦之下，人口拥挤，生态环境逐渐破坏，土地变得贫瘠，粮食产量远远不能满足当地居民，特别是城市居民的需求。相比之下，长江流域，特别是长江下游地区，经过从三国，东晋到南朝的开发，逐渐成为重要的产粮区。

    因此，为了解决北方，特别是作为隋唐政治中心——关中地区的粮食需求，隋朝的两任皇帝挖掘了全国性的大运河。

    北宋末年，中国人口首次过亿。

    • 在人口繁衍的同时，中国的耕地面积和农业总产量也相应提高了。
    • 到了宋代，政府不再像隋那样限制农民的流动，并且还给予了佃户人身自由。只要租种地主土地的人和地主结算清楚地租，余下的收入都可以归他自己所有，同时地主也不得干预农民经营。
    • 有了坚实的农业基础后，中国的采矿业在唐宋时期也得到迅速的发展。
    • 而金属的冶炼和铸造，需要大量的煤炭，这又推动了唐宋之际，煤炭业的兴盛。
    • 另外唐宋时期中国水平最高的产业是陶瓷制造业和纺织业，它们在当时世界上都处于领先水平。

    唐三彩的釉色其实有很多种，但各种釉色彩陶都以黄、绿、褐（或绿、赭、蓝）三色为主，所以又叫“三彩”

    工商业和贸易的繁荣导致了宋代纸币的出现。在北宋初年，四川地区出现了世界上最早的纸币——交子，这是一次货币革命，它一方面给民众带来了巨大的便利性，另一方面也引发了通货膨胀的问题。

    # 科技发展

    在唐代，中国科技的成就主要体现在算术天文、医学和工程技术这三个领域。

    唐高宗即位后，新添了算学馆，不仅教授和研究算术，而且参与天文观察。数学的普及为唐代历法的修订打下了基础。

    此外，唐代在医学上也有很大的进步。中国传统的医学，也就是常说的中医，形成于汉代，但是成为一个系统性的学科，并且指导民众按照中医的思想生活是在唐代。这一方面是因为唐代出了一些名医，包括孙思邈、甄权、孟诜等人。另一方面也是因为朝廷比较重视医疗，在长安和各州县都设有官方医疗机构。

    唐代另一个让世人瞩目的科技成就是火药的发明。虽然火药被发明的时间至今还无法确定，但是学术界一般认为火药发明于7世纪的中国，是当时中国术士炼制长生不老药时意外发现得到的副产品。
