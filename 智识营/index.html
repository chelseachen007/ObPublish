    # 作业

    # 第一题

    1: 预测性编码； 2：深度加工；3：间隔测试； 4：框架思维 ；5：敏感化；6：大脑的投入量

    # 第二题

    为什么读书的时候觉得都记住了，但过了一段时间，却什么都回忆不起来？请回答产生这种现象【最可能】的原因？

    人类的大脑在几千上万年的进化中，产生了一种节能机制，这种机制会清除掉一些无用的信息。信息在大脑中的构成是一个个神经元，如果我们读书的时候没有带着问题去读，信息就仿佛一个孤立的神经元，没有产生链接。而之后你又没有经常去提取和回忆，加强提取强度，久而久之就会被清除出去。

    # 第三题

    在预测性编码一节我们讲过：人们之所以更容易记住故事，一大原因是因为故事的模式往往符合我们某些预测框架。那么，除了这个原因之外，是否还有别的原因？请用深度加工理论解释

    深度加工理论里有一条，讲的是建立联系，他指的是一个信息点跟我们知识体系里其他信息的链接要多且不长，而故事就是这样子设计的，他选取的主人公或背景或其他的一些信息，往往都是我们耳熟能详的一些常识，这些信息与我们的知识体系中的知识点路径短能快速建立联系，我们得准确的预知到下一步会是什么。但他又会在一些点上巧妙的设置反转，让我们的预测被打断，让我们神经对这个信息的反应加强，达到对这个故事影响深刻的目的。
