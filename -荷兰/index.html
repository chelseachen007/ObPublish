🇳🇱 荷兰

最近编辑于 Jan 4, 2024

    荷兰的正式国名其实应该是“尼德兰”，它在近代刚开始时指欧洲西北部的低地地区，既包括今天的荷兰，也包括今天比利时的部分地区。荷兰只是当时尼德兰地区的一个省

    # 早期

    早期的荷兰从很多角度来看，都不具备成为世界大国的条件。

    首先，它拥有着不适合种植和生活的地理和气候条件。荷兰有约三分之一的陆地在海平面上一米左右，还有很大一部分领土则在海平面以下。而且它还处于北纬50度左右的高纬度地区，接近我国黑龙江省漠河的纬度。那里的冬季，气候阴冷潮湿。由于日照短，谷物在那里也很难生长。

    其次，荷兰还缺乏可发展经济的自然资源。那里除了鲱鱼，几乎没有其他自然资源。当然，那个时候北海地区深埋的石油还没有被发现。

    最后，它的文明发展历史也不长。因为远离古代的几大主要文明，尼德兰地区的文明直到12世纪左右才有了较大的发展

    # 文艺复兴时期

    # 捕鱼业

    因为荷兰靠海，每年鲱鱼都要成群结队地游到荷兰周围的北海地区，当地人于是从事起了打捞和贩卖鲱鱼的生意。又因为他们发明了快速清洗和腌制鲱鱼的方法，这种方法能将原来一两天就会变质的鲱鱼，保存一年之久。
    荷兰人则可以将它们卖到欧洲大部分地区，这就让荷兰人垄断了西欧的渔业市场。

    # 海上运输业

    由于地处高纬度的寒冷地区，生活不易，荷兰人为了生存大多都很勤劳。他们在不能打渔的季节，就用渔船在北欧和西欧为他人运输货物。后来荷兰人就有了“海上马车夫”的名声

    # 手工业

    靠渔业和运输业攒下一些钱后，荷兰人就开始发展手工业。因为当地缺乏工业原材料，他们就进口棉花和羊毛，加工成布匹和毛呢出口，赚取加工的利润

    # 独立战争

    长期经商让荷兰人变得很务实，对政治没有太多兴趣，他们甚至不怎么关心统治者是谁，只要新的统治者不干涉他们挣钱。这导致荷兰从一开始就缺乏国家的独立性

    直到查理五世开始向荷兰人征重税，动了他们的经济利益，荷兰人就不干了。他们先是和西班牙人和平交涉，希望获得商业的主导权和宗教自由，并且希望西班牙国王给予市民们充分的权利，市民们也站出来反对重税。
    （ 除了经济上的原因，荷兰人反抗西班牙人还有一个原因，那就是荷兰人大多信奉加尔文派的新教，但是西班牙人却笃信天主教，双方经常因为宗教问题起冲突。

    荷兰人不得不通过战争的方式反抗西班牙人，来捍卫自己的利益。因此，双方从1568年到1648年，打了80年的仗，史称“八十年战争”。

    # 大航海时代

    在西班牙鼎盛时期，它获得了对尼德兰的统治权，不过经过几十年的独立运动和战争，荷兰终于摆脱了西班牙的统治，变成了近代早期的资本主义国家。靠着制造业和贸易，荷兰迅速强大起来，并且加入了对世界海上航线和贸易的争夺战。

    经过以上这些努力，荷兰后来发展成为了世界上第一个真正意义上的资本主义国家。 🇬🇧 英国在光荣革命后进入到资本主义时代，比荷兰晚了40多年

    1602年，荷兰成立了著名的东印度公司。这个组织虽然叫做公司，但是和今天的公司大不相同，因为它不仅拥有一只强大的武装力量，而且可以自行发行货币，并有权代表国家和其他国家正式签订条约

    更为特别的是，东印度公司在荷兰的六个城市设有办事处，它的股东包括荷兰政府和大部分国民。靠国民支持的荷兰式的大航海和靠王室支持的葡萄牙、西班牙大航海是有很多差别的.

    荷兰人在大航海中还有两次壮举。

    # 发现澳大利亚

    一是发现了澳大利亚。 1606年，荷兰人威廉∙詹茨（Willem Janszoon，1570～1630）第一次发现了这块大陆，并且在约克角半岛的西岸第一次登陆。至此，世界上除南极以外的大陆都被发现了。

    # 北极探险

    荷兰人的另一个壮举就是对北极的探险，当然探险的目的是寻找通往亚洲最近的航道，而不是科学考察。他们的想法是，既然地球是圆的，从理论上讲，往任何方向走都能到达亚洲。于是，地处北半球高纬度国家的荷兰人就开始打往北走的主意了。

    从1594年开始，荷兰探险家和航海家巴伦支(Willem Barents，1550－1597)对北极进行了三次考察，希望通过北极到达亚洲。不过遗憾的是，由于极寒天气和浮冰的阻挡，他并没有成功。

    值得一提的是，在1596年的第三次探险中，巴伦支的船队被困在北极的冰层里长达八、九个月，船员们有很多饿死冻死了，但是也没有动船队所运送的货物。这件事在西方近代史上树立了信守契约责任的榜样。

    # 东印度公司

    伊比利亚半岛航海的资金大部分是靠国王支持的，但是荷兰并没有强大的王权，在航海事业起步初期，资金来源主要是靠市民的支持。

    荷兰不缺乏好的水手和精明的商人，所欠缺的就是进行远洋贸易所需要的大量资金。而作为一个刚刚独立的国家，荷兰政府并没有多少钱可以投资。但是由于长期采取宽松的政策，藏富于民，市民们还是很富庶的。有钱之后，荷兰人发明了一种集资的方法，也就是由市民们出钱成立一个股份公司，参股的市民就成为了公司的股东，将来可以从公司的利润中分红。

    这样，公司的股东，也就是投资的市民，和经营管理层，也就是愿意冒险的商人，就分开了。

    除此之外，东印度公司还有许多前无古人后无来者的特殊之处。

    • 首先，它具有几项作为国家才有的权利。当初荷兰政府为了支持这家公司，发给了它一纸特许状。有了这个特许状，东印度公司可以拥有自己的军队，在海外建立殖民地，并且有权代表荷兰和外国缔结条约。
    • 其次，东印度公司的股东，或者说股民数量非常庞大，就连当时一些有钱人家的仆人也持有这个公司的股票。

    # 荷兰人为什么敢于把自己的积蓄投入这项冒险呢？

    • 首先，荷兰人本身就比较喜欢冒险和投资，著名的金融街区华尔街就是荷兰殖民者首先在美国建立起来的。
    • 其次，东印度公司有荷兰政府的背书，这在一定程度上让东印度公司更容易赢得人民的信任。
    • 最重要的是，荷兰商人有良好的信誉，并且是世界上最早遵守信托责任的群体。例子：阿姆斯特丹银行

    # 为什么崛起，而西班牙和葡萄牙衰落

    # 利益分配差异

    西班牙和葡萄牙的船队是各自独立核算，这就导致了投资者每一次的收益并不稳定。在这样的氛围里，很多生意是一锤子买卖，比如这次在亚洲捞到一票，投资人可能就不再去做这种高风险的投资了。

    而东印度公司则不同，它追求的是一个长远的买卖。在东印度公司成立的头几年里，公司并没有派发股息，而是将全部的利润投入到扩大对外贸易和修建基础设施上。他们在世界各处修建码头，仓库，船坞，同时大量造船扩大船队。
    在确立了全球贸易的主导地位后，公司才给股东支付了第一次红利，并且以后每年都向国民和政府支付高额的红利。在世界的历史上，不断地证明着一个真理，一个维护少数人富裕的国家，虽然可以强大一时，但是很难持久。

    荷兰人为什么能接受几年不分红？这其实不全是因为他们觉悟高和有远见，这里面的关键在于，荷兰人发明了一种新的资本运作体制——证券交易。
