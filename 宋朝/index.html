宋朝

最近编辑于 Dec 28, 2023

    由于宋王朝的面积远逊于唐帝国最强盛的时期，甚至比不上两汉，因此在过去很多人对宋留下了积贫积弱的印象。但是宋代的人口并不比唐代少，而且如果要论经济总量和政府的财政收入，宋朝甚至远高于唐代。也因此，在今天很多历史学家看来，宋代才是中华农耕文明的顶点，也是上至官员、下至百姓生活最好的朝代。

    # 宋对比唐朝的变化

    • 世家豪族退出历史的舞台，皇权进一步集中，中国政治走向平民化。 自宋以降，科举进入仕途几乎成为了汉族子弟进入仕途唯一的通道，即使是当朝达官贵人的子弟也不例外。这种体制大大增强了士人对皇权的认可，他们会以天子门生和孔孟之徒自居。

    • 重文轻武。这种风气一直持续到清末。

      建立宋朝的赵匡胤自己靠军事政变夺位，他总结了自中晚唐以来军阀割据的教训，制定了很多限制武将权力的措施，比如由皇帝直接掌控战斗力较强的禁军，刻意压低武将的地位，由文官管理军事等等

      • 在对外关系上，宋朝是中国历史上少有的尽可能以岁币换和平的政权。
      • 在经济上，宋王朝以农业为主，但是并不过分抑制商业，这让宋王朝成为了中国历史上最富庶的朝代。而其北方的辽、金等政权，也不单纯的是以畜牧业为主的北方族群，而是开始了自己的农业生产活动。

    # 科技

    唐宋时期，除了文化出版、唐诗宋词，这个时代同时也是中国古代科技发展的顶峰。

    李约瑟在其著作《中国科学技术史》的导论中讲，“每当人们在中国的文献中查找一种具体的科技史料时，往往会发现它的焦点在宋代，不管在应用科学方面，还是纯粹科学方面都是如此。”这种说法是符合历史事实的。

    # 数学

    首先是数学。北宋的数学家贾宪，是世界上最早提出二项式分解计算公式的数学家之一。这项成就在很长的时间里被归功于南宋数学家杨辉的发明，因此这项成就在过去被称为“杨辉三角形”，后来在大英博物馆所藏的《永乐大典》中发现贾宪的《释锁算术》中已经有了这项发现

    到了南宋，数学家秦九韶系统地解决了数论中著名的中国余数问题，提出了初等数论中关于余数的基础理论。秦九韶著有《数书九章》一书，它是一本实用数学大全，系统讲述了算术、几何学、解方程和天文学的知识。

    # 冶金术

    • 一项是冶炼异质结构钢材的方法。通常的钢材内部都是同质结构，很难兼顾强度和硬度，硬度大的钢常常太脆，强度低，强度大的钢材往往不够硬，而异质结构钢材可以克服这个问题，但是需要很高的冶炼技术。
    • 另一个发明是炼钢熔炉，它们是现代熔炉的前身，通过鼓冷风反复锻造，来达到脱碳的目的。
      冶炼技术的改进让宋代的钢铁产量剧增。

    # 其他

    • 毕昇的活字印刷术
    • 宋朝人还发明了用于指示方位的指南车，这是指南针在中国最早的应用。
    • 四大发明中的另一项发明火药，在宋代也被用于军事。宋朝人发明了原始的火枪，以竹筒为枪管的火铳。
    • 宋代人发明了复闸形式的运河船闸，今天巴拿马运河使用的就是这种船闸。有了船闸，就能帮助运河翻越高地，提高运输能力。
