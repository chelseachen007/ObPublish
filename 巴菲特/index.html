    # 巴菲特

    # 巴菲特的投资

    # 2024年宣布“无限期持有”

    # 西方石油公司

    美国的西方石油公司，伯克希尔特有它27.8%的普通股和一大笔的未来认股权证。

    巴菲特一边成了西方石油持股接近30%的大股东，另一边却说，伯克希尔公司无意收购或管理西方石油公司，只是看好它在美国拥有的大量石油和天然气资产，以及它的碳捕集技术。

    # 能源公司思路转变

    • 在以前，能源和资源都属于周期性行业，需求高了就扩产，需求少了就减产
    • 但现在全球上游企业们的判断是，页岩油革命不可逆、新能源转型不可逆，经济下滑不知道还要持续多久，反正自己未来是要被替代掉的，今天再为未来做长远投资就不值得了。
    • 于是，全球资源企业们不约而同转向了同一种心态：不再投资扩产，保持现有产能的利润增长和现金流增长就可以了。让自Ï己和股东过得开心点。

    能源公司们这种“自己要被替代掉”的预期虽然 还没有成为现实，但它们心态一变，已经导致全球传统能源和大宗商品的供给收缩得比 需求收缩得更快了。

    这种收缩产能的局面下，资源企业们突然 跳出了传统的产业周期，变成了值钱的现金奶牛。
    稳定派息的现金奶牛，这正是巴菲特的最爱。

    # 日本五大商社

    他把日本的伊藤忠、丸红、三菱、三井和住友公司整合起来，统称为“日本五大商社”，伯克希尔分别持有各家大概9%的股份。

    巴菲特说，伯克希尔会继续长期被动地持有日本五大商社股份。

    • 什么是被动持有？意思是当纯财务投资者，拿着股份，但不主动参与公司的管理，也不试图影响公司的运营方向，只是图它的股息或资本增值。

    # 买公司就是买资产

    首先，巴菲特选择在日本投资，从表面上看投的是日本商社，本质上，是想持有过去40年日本在海外累积起来的全球资产

    • 1990年代，日本经济泡沫破灭，进入失去的三十年，当时日本为了支撑经济选择大举全球化。
    • 五大商社就带着使命在全球加速进行投资布局，尤其是买了大量自然资源
    • 比如金属矿、煤矿和各类能源项目，所以它们如今资产当中的超过一半是能源和矿产
    • 而到了今天，美国想要重新划定世界经济版图，重建一个不带中国玩的朋友圈那么日本拥有的全球资源、物流、贸易和产业链体系，在美国看来正好能跟它的金融、资本、技术、军事形成强大的互补。

    # 套息

    其次，巴菲特买入五大商社的股份，是在执行一种风险非常小的套息策略

    日本为了刺激经济，从1999年开始实施零利率政策，2016年又把政策利率降至负区间，负利率一路维持到了2023年。

    • 美国这边，2022年3月美联储开始激进加息之后，美元日元之间就出现了利差交易
    • 大量资本都去抛日元、买美元。
    • 日元不值钱，换成美元高息资产，能吃到高利息。
    • 但巴菲特在过去几年却反向行之，借入了大量的日元。
    • 他在2019年到2023年期间，总共发债借入了1.3万亿日元，把伯克希尔变成了日元债券的最大海外发行者之一。

    借钱干嘛呢？就是买五大商社的股份。借日元买日股。

    2023年4月，巴菲特以92岁高龄，不远万里亲自到访日本

    • 这之后，巴菲特就向市场高调宣布增持五大商社，这一宣布带动了全世界对日本股市的信心。
    • 日经指数已经一路走高到了39000点，创下了33年来的最大涨幅，收复了“失去的三十年”。
    • 他们聊了什么，没有向外透露，付鹏的推测是，双方议题之一，就是巴菲特提供资金支持五大商社向新兴产业的转型，条件是五大商社要加大分红派息和回购的比例。
    • 在日本的负利率环境下，日元的融资利率只有0.5%，哪怕加上其它一些发行费用，伯克希尔发行日元债的融资成本大概在0.6%，四舍五入资金成本接近为0。
    • 日本经济哪怕不能复苏，五大商社只要加大分红，就能源源不断给伯克希尔注入现金流
    • 只要持续回购自身股票，就可以抵消经济基本面的下行、稳住股价。
