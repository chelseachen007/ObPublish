如何做出海产品

最近编辑于 Aug 2, 2023

    主流论调是你要自己有需求，才能对需求了解的深刻，才能开发好，
    但很多人误解了这句话的意思，总是从自身身上想到一些自动化的需求。这样很容易陷入拿着锤子去找钉子的情况。

    想做好出海产品，第一步学会抄，第二步学会优化，第三步才是进行自己的微创新

    # 怎么抄

    通过投流网站寻找热点，一般有起色的网站都会广告，至少会投一下自己的品牌词，比如chatbase.co，品牌词就是chatbase，如果你不投，别人发现了，别人可能会投你这个词，然后将流量弄到自己的网站上，这便是所谓的品牌词捡漏投法,详情看Mediabuy

    但用ahrefs看见的数据是片面的，我们结合similarweb看看。

    简而言之，判断能不能抄这个操作足够熟练时，你就有一定的能力去判断需求真伪了

    # 获取流量

    做产品前，还是要想想流量来源。

    一般而已，海外产品开发完，会在 producthunt.com、 indiehackers.com、 twitter.com上发帖。但这种流量属于慢且不精准的流量。

    一般海外的SaaS产品流量来源就几个方面：自然搜索词、外链、广告投流、红人合作。

    # 多看且有方法的看

    # 什么样的产品你是想看的？

    最近6个月流量起来的，有用户付费的产品，然后你就可以结合数据分析工具，将这批产品拉出来去看。

    # 判断是否值得去做

    可以下一些数据定义，比如竞品不超过10个，最老的竞品运营时间不超1年，流量刚起来的等等。

    # 看那些渠道

    • Indie Hackers: 自由职业者论坛
    • 生财有术
    • 各种大 V，我还没关注这些

    # 不要过于追求技术

    做一个创业者，读源码就属于高付出低回报的事，用自己最熟悉的技术栈，最快最小的完成客户的需求最好，当然要注意一些可复用组件和代码的积累。
    像很多需求，其实不需要很多技术，像花开哥服务快手地推的人群，提供了一些图片上传、短链等服务，都是一些简单的技术，但能带来很多收益。所以，技术人一定不要痴迷技术，技术最终都是为了服务于业务。

    # 程序员如何打造受人喜爱的出海工具？

    打开 WebsiteTraffic-CheckandAnalyzeAnyWebsiteSimilarweb 。

    1. 选择一个你熟悉的网站去看数据
    2. 去看热门受众兴趣点和竞对网站
    3. 通过一个域名，找到10个以上的新域名
    4. 以此类推，不断发现新域名新工具站
