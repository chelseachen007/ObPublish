    # 朱元璋

    元朝统治不把人当人，从赋税要徭役，干不干活都要收税

    1344 年转折点来了

    • 黄河泛滥了，沿岸山东河南几十万人沦为难民。
    • 淮河沿岸遭遇严重瘟疫和旱灾。

    灾难到来后，四月初六朱重八的父亲饿死，初九大哥饿死，十二日大哥长子饿死，二十二日母亲饿死。

    朱重八的姐姐已经出嫁，三哥去做了倒插门。除了朱重八的二哥，这个家庭已经没有了其他成员。

    为了有饭吃，朱重八决定去附近的皇觉寺当和尚。
    但是他入寺仅五十余天后，由于饥荒过于严重，所有的和尚都要出去化缘。（也就是出去讨饭

    在讨饭的时候，他仔细研究了淮西的地理、山脉、风土人情，他开阔了视野，丰富了见识，认识了很多豪杰（实际上也是讨饭者）。此时，他还有了自己的宗教信仰——明教，他相信当黑暗笼罩大地的时候，伟大的弥勒佛一定会降世的。

    现在看来，脱脱坚决要求治黄河的愿望是好的，然而他不懂得那些反对的人的苦心，元朝那些腐败到极点的官吏也是他所不了解的。现在他终于要尝到苦果了。

    当民工们挖到山东时，他们从河道下挖出了一个一只眼睛的石人，背部刻着：石人一只眼，挑动黄河天下反。民工们突然发现，这正是他们在工地上传唱了几年的歌词。于是人心思动。

    公元1351年，韩山童、刘福通第一个举起了反抗元朝封建统治的大旗。

    本来朱元璋还不准备造反，直到有一天，幼时的朋友汤和当上了起义军的千户，希望他能一起来共图富贵。到这里，朱元璋仍不准备造反，但是后来得到消息背后有人告发他了，他找到周德兴算了一卦，走上了造反的路。

    投奔了郭子兴后，凭借他作战勇敢、很有计谋，处事冷静，思虑深远（注意这个特点），还很讲义气，有危险的时候第一个上，这一切都让他有了崇高的威信。这让他二个月就当上了九人长

    也就在此时，朱重八将他的名字改成了朱元璋。所谓璋，是一种尖锐的玉器，这个朱元璋实际上就是诛元璋，朱重八把他自己比成诛灭元朝的利器，而这一利器正是元朝的统治者自己铸造出来的。

    郭子兴也越来越讨厌朱元璋，原因很简单，朱元璋比他强。对于郭子兴这样一个性情暴躁、不能容人的统帅来说，他是不能容忍一个可能取代他地位的人在身边的。终于有一天，他把朱元璋关了起来。
    善良的马姑娘为了救朱元璋，便把刚烙好的烙饼揣在怀中，到牢中探望朱元璋时送给他吃，每次胸口都会烫伤，但每次都送。

    后来被放出来后，朱元璋决心远离这些鼠目寸光的人，申请带兵出征，攻打重兵把守的定远。
    此后，又连续攻击怀远、安奉、含山、虹县，四战四胜，锐不可当！随后，朱元璋来到了家乡钟离，召集了 700 人左右的直属部队。

    当朱元璋再次回到濠州的时候，他已经完全明白了自己的前途所在，所以他向郭子兴提出了辞职。郭子兴非常高兴，这个讨厌的人终于可以走得远远的了。

    朱元璋在出发前，又做了一件出人意料的事，他从自己的七百人中重新挑选了二十四个人，然后将其余的人都给了郭子兴

    随后，朱元璋在滁州收编了一个山寨，打下了一个 2 万人的军队，并严肃纪律，训练有加，相当于有了第一桶金。
    在这期间有了冯国用、冯国胜的投奔，遇到了一个书生李善长，又迎来了三个重要的人，分别是他的侄子朱文正、姐夫李贞和外甥李文忠。

    随后，郭子兴被赶出濠州，投奔朱元璋，朱元璋不计前嫌，把统帅和三万精兵交给了郭子兴，这一举动折服了所有想害他的人。
    但郭子兴气量太小，后面没杀掉孙德崖留下心病，被气死了。

    # 霸业开始

    和州太小了，朱元璋随后把目光放到了集庆（南京）。

    南京 ，紫金山纵横南北，恰似巨龙潜伏，而石头山则临江陡峭，如虎盘踞，这就是南京龙蟠虎踞的来历。是具有王气之地。
    附近不但是重要的粮食产区，还兼着商业中心的作用，最重要的是，这里有运河之利，在那个从北京走到南京要几个月的年代，水路实在是太重要了。

    但攻打南京需要度过长江，朱元璋没有水军。这时，一个叫俞通海的海盗到来解决了这个问题。

    由于船只太差，而且过于小看集庆的城防，朱元璋于至正十五年(1355)八月和九月连续两次攻击集庆，都被元军击败。然而失败对朱元璋来说并不一定是坏事，因为在这两次战斗中，郭天叙和张天祐都战死了，朱元璋顺理成章地成为了都元帅。
    第二年(1356)，朱元璋亲自带兵分三路进攻集庆，用了十天时间攻破了集庆，并改集庆为应天。

    从历史大势上看，他的形势并不乐观，自古占据北方即有天时地利，中国地势由北向南倾斜，由南方起兵进攻北方而最后获得胜利，少有先例。

    我们介绍一下朱元璋的邻居们。住在他东边镇江的是元朝军队，而住东南方平江（今江苏苏州）的是张士诚，东北面的是张明鉴的起义军，南面是元将八思尔不花（名字很有特点），西面是徐寿辉。

    # 律法

    朱元璋制定了《大明律》，并规定了五种刑罚，分别是笞、杖、徒、流、死。
    而在明初的普法教育中，最重要的并不是《大明律》，而是一本叫做《大诰》的书。所谓《大诰》是朱元璋采集一万多个罪犯的案例，将其犯罪过程、处罚方式编写成册，广泛散发，人手一册。根据法理学的分类，《大诰》采用了案例，应属于判例法，这么看来朱元璋还颇有点英美法系的倾向

    人去世是不能随便说死的，要先看人的身份，具体规定如下：皇帝死称崩，公侯贵戚死称薨，大臣死称卒，士死称不禄，庶人死才能称死。

    当时的人按职业可大致划分为：民户、军户、匠户。其中民户包括儒户、医户等，军户包括校尉、力士、弓兵、铺兵等，匠户分为工匠户、厨役户、裁缝户等。

    这些户的划分是很严格的，主要是为了用人方便，要打仗就召集军户，要修工程就召集匠户。看上去似乎也没有什么问题，但其实缺陷很大。

    这样的划分是不科学的，不但民户军户这些大户之间不能转，同一户内不同的职业也不能转，万一打仗时要召集弓兵户，偏偏这些人从小没有练过拉弓，那也没办法了，每人给一把弓就上吧。这样的军队战斗力是不能保证的，又如召集医户，如果召来的都是一些不懂医术的，那可就要出人命了。

    这是老百姓，官员们也不轻松，他们衣服穿着和颜色搭配、用料、图案都有着严格的规定，如违反，是要杀头的。

    是的，朱元璋要建立的是一个等级分明、秩序严谨、近乎僵化的社会结构，在这个结构中，农民只能种地，商人只能经商，官员按照规定干好自己的工作，无论谁都不能越界。军户、民户、匠户的划分

    # 建文帝朱允炆

    二十一岁的少年登基后，发现有二十四个叔叔拥有武装，其中又以保卫边疆的九个最为突出，从东北到西北分别是辽王、宁王、燕王、谷王、代王、晋王、秦王、庆王和肃王。

    其中燕王和晋王势力最大，他们各自带有十余万军队，可谓兵势强大。但这二位还不是九王中最生猛的，公认的打仗第一强人是宁王，此人“带甲八万，革车六千”，看似兵力没有燕王和晋王多，但他手下却有一支当年最为强大的武装——朵颜三卫。（也就是蒙古人

    建文帝登基后，立刻召回方孝孺，任命为翰林侍讲，并提升齐泰为兵部尚书、黄子澄为翰林学士。
    但是这三个有几个共同特点：都是饱读诗书，都是文人，都有远大理想，都是书呆子。

    最终选择烧了皇宫，但人不知踪影，这给朱棣留下了大难题。又不能明面上找和逮捕，只能派人暗中查访建文帝的下落，具体的查访工作由两个人去做，这两个人寻访的路线也不同，分别是本土和海外。这两个人的名字，一个叫胡濙，另一个叫郑和。

    # 明宣宗朱瞻基

    年号宣德

    于谦

    朱瞻基的统治时间并不长，只有十年，加上他父亲的统治时间，也只有十一年，但他和他父亲统治的这短短十一年，却被后代史学家公认为是堪与“文景之治”相比的“仁宣之治”，是中国历史上的盛世。

    何以有如此之高的评价，盛世何来？来自休养生息，清静养民。其实封建社会的老百姓们自我发展能力并不差，你就算不对他进行思想教育，他也知道自己要吃饭，要挣钱，要过好日子，只要官府别天天加收田赋，征收徭役，给这些不堪重负的人们一点喘息之机，他们是会努力工作的。

    首席活太师，所谓荣誉称号很多时候都是送给死人的，而能够在死后混到太师、太傅、太保这三个的，也是十分厉害的人。

    # 明英宗朱祁镇

    年号正统

    宣德十年(1435)，一代英主朱瞻基经抢救无效死亡，年仅三十八岁。年仅 9 岁的朱祁镇，并有三杨、张辅、胡濙五人担任顾命大臣

    另外还有个重要的女性，朱祁镇的祖母–张太皇太后。
    十一年前，她是张皇后，十年前，她是张太后，现在，她是张太皇太后。

    他跟朱祁镇说，你做任何决定都要经过五位顾命大臣的赞同，并且追着王振打压了七年。

    王振在面对三杨谦恭的表象之下，却不断地拉帮结伙，扩大自己的势力，他利用司礼监的权力安插自己的侄子王山为锦衣卫同知，并广结党羽，控制朝臣。

    正统七年(1442)十月，历经四朝的张太后离开了人间，三杨中的杨荣也已经去世，而剩下的杨士奇和杨溥也已年老多病。

    王振从此大权独揽，广结同党，不但控制了锦衣卫，还收了很多属下，其中不乏饱学之辈，圣人门徒，

    他排除异己，利用杨士奇儿子杀人的事件，攻击他教子无方，最终打垮了这位四朝老臣，之后他又陆续诬陷户部尚书刘中、祭酒李时勉等不服从他的大臣，并把他们赶出了京城。

    此时的王振，内得皇帝信任，外有打手帮忙，独掌大权，鱼肉百官，可谓风光无限，成为了明朝开国以来最有权势的太监。

    有钱有势之后，王振就想着建功立业，名留青史了

    # 兵败土木堡

    为什么也先不直取京城
    二十万大军毁于一旦，无数文官武将战死，最为精锐的三大营全军覆没，京城已经不堪一击。

    众人乱成一锅粥，吏部尚书王直指出一个更重要的问题：皇帝死了没?
    万一把皇帝当成死人注销了户口和皇籍，另外立了皇帝，过两天他自己屁颠屁颠地回来了，你还要脑袋不要？

    不用担心，也先抓了朱祁镇，也怕你重新立个皇帝，那手上这个不是就没有作用了，于是第二天就加急让他们知道了：你们的好皇帝还活着。

    皇帝当人质真的是件棘手的事，给钱也不一定能换回来，说不定还上瘾每年收一次保护费。
    打？说不定拿皇帝当肉盾，谁敢打，打死了就是诛九族的事。更何况现在京城里都是老弱残兵士气低弱。

    朱祁镇她立刻把后宫的所有金银珠宝全部派人送到也先的军营里，希望能够赎回丈夫。但这显然是肉包子打狗，一去不回。

    文官们主张南迁，毕竟流水的皇帝，铁打的打工人，官还是要继续当的。

    这一幕和三百二十年前的北宋多么像，金兵渡过黄河，金将完颜宗望押着被俘的宋徽宗、宋钦宗和赵氏皇子后妃、宫女四百余人及其掠夺的大量金银财宝回朝，大臣们毫无战意，主张投降。

    但他们又不一样，只因为大明有于谦，一声怒吼：“建议南迁之人，该杀！”

    于是传奇开始，北京保卫战

    # 还乡团的斗争

    天顺八年(1464)，即将离世的朱祁镇，对朱见深说出了最后的遗愿：“自高皇帝以来，但逢帝崩，总要后宫多人殉葬，我不忍心这样做，我死后不要殉葬，你要记住，今后也不能再有这样的事情！”

    朱祁镇有一种他的先辈们所不具备（或不愿意具备）的能力——理解别人的痛苦。

    当年明月对其的评价是：他是一个好人，却不是个好皇帝。
