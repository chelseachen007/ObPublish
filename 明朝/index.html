    # 律法

    朱元璋制定了《大明律》，并规定了五种刑罚，分别是笞、杖、徒、流、死。
    而在明初的普法教育中，最重要的并不是《大明律》，而是一本叫做《大诰》的书。所谓《大诰》是朱元璋采集一万多个罪犯的案例，将其犯罪过程、处罚方式编写成册，广泛散发，人手一册。根据法理学的分类，《大诰》采用了案例，应属于判例法，这么看来朱元璋还颇有点英美法系的倾向

    人去世是不能随便说死的，要先看人的身份，具体规定如下：皇帝死称崩，公侯贵戚死称薨，大臣死称卒，士死称不禄，庶人死才能称死。

    当时的人按职业可大致划分为：民户、军户、匠户。其中民户包括儒户、医户等，军户包括校尉、力士、弓兵、铺兵等，匠户分为工匠户、厨役户、裁缝户等。

    这些户的划分是很严格的，主要是为了用人方便，要打仗就召集军户，要修工程就召集匠户。看上去似乎也没有什么问题，但其实缺陷很大。

    这样的划分是不科学的，不但民户军户这些大户之间不能转，同一户内不同的职业也不能转，万一打仗时要召集弓兵户，偏偏这些人从小没有练过拉弓，那也没办法了，每人给一把弓就上吧。这样的军队战斗力是不能保证的，又如召集医户，如果召来的都是一些不懂医术的，那可就要出人命了。

    这是老百姓，官员们也不轻松，他们衣服穿着和颜色搭配、用料、图案都有着严格的规定，如违反，是要杀头的。

    是的，朱元璋要建立的是一个等级分明、秩序严谨、近乎僵化的社会结构，在这个结构中，农民只能种地，商人只能经商，官员按照规定干好自己的工作，无论谁都不能越界。军户、民户、匠户的划分

    # 建文帝朱允炆

    二十一岁的少年登基后，发现有二十四个叔叔拥有武装，其中又以保卫边疆的九个最为突出，从东北到西北分别是辽王、宁王、燕王、谷王、代王、晋王、秦王、庆王和肃王。

    其中燕王和晋王势力最大，他们各自带有十余万军队，可谓兵势强大。但这二位还不是九王中最生猛的，公认的打仗第一强人是宁王，此人“带甲八万，革车六千”，看似兵力没有燕王和晋王多，但他手下却有一支当年最为强大的武装——朵颜三卫。（也就是蒙古人

    建文帝登基后，立刻召回方孝孺，任命为翰林侍讲，并提升齐泰为兵部尚书、黄子澄为翰林学士。
    但是这三个有几个共同特点：都是饱读诗书，都是文人，都有远大理想，都是书呆子。

    最终选择烧了皇宫，但人不知踪影，这给朱棣留下了大难题。又不能明面上找和逮捕，只能派人暗中查访建文帝的下落，具体的查访工作由两个人去做，这两个人寻访的路线也不同，分别是本土和海外。这两个人的名字，一个叫胡濙，另一个叫郑和。

    # 明宣宗朱瞻基

    年号宣德

    朱瞻基的统治时间并不长，只有十年，加上他父亲的统治时间，也只有十一年，但他和他父亲统治的这短短十一年，却被后代史学家公认为是堪与“文景之治”相比的“仁宣之治”，是中国历史上的盛世。

    何以有如此之高的评价，盛世何来？来自休养生息，清静养民。其实封建社会的老百姓们自我发展能力并不差，你就算不对他进行思想教育，他也知道自己要吃饭，要挣钱，要过好日子，只要官府别天天加收田赋，征收徭役，给这些不堪重负的人们一点喘息之机，他们是会努力工作的。

    他教太监读书，来帮自己分担政务和对抗文官集团，这可以说是对抗的必然吧。但宦官集团感觉是明朝乱的一大原因，朱瞻基可以说是打开这个大门吧。

    # 明英宗朱祁镇

    1436,年号正统，不幸的是他 9 岁继位，幸运的是明宣宗留下了几个好大臣。但都是老人，没几年就相继去世，使得宦官当权，这其实就是宦官读书的弊端，你得管得住宦官，而且宦官不像文人集团有节气。
    所以第一个权倾天下的宦官诞生了,这也是明朝悲剧的开始。

    # 还乡团的斗争

    天顺八年(1464)，即将离世的朱祁镇，对朱见深说出了最后的遗愿：“自高皇帝以来，但逢帝崩，总要后宫多人殉葬，我不忍心这样做，我死后不要殉葬，你要记住，今后也不能再有这样的事情！”

    朱祁镇有一种他的先辈们所不具备（或不愿意具备）的能力——理解别人的痛苦。

    当年明月对其的评价是：他是一个好人，却不是个好皇帝。

    好人不太适合当一个最高统治者，就像三体（全集）中的程心一样，真的是一件很矛盾的事。

    # 明光宗朱常洛

    万历四十八年(1620)八月一日，定年号为泰昌。
    明神宗长子但是明神宗一直喜欢的是三皇子朱常洵，所以立太子一事用了二十年。

    可几乎所有的人都没有想到，这个年号，竟然没能用上。因为朱常洛活了三十八年，明光宗却只活了一个月。

    # 南明

    # 形成原因

    “民穷”的根本原因不在国家的赋税过重，而端在法律的腐败和政府的低能。国家的税率低，受惠者并非农民，只是鼓励了大小地主加重剥削以及官僚乡里额外加征。

    明朝采取严格的中央集权，施政方针不着眼于提倡扶助先进的经济，以增益全国财富，而是保护落后的经济，以均衡的姿态维持王朝的安全。这种情形，在世界史中实属罕见，在中国历史中也以明代为甚，而其始作俑者厥为明太祖朱元璋。

    黄仁宇认为中国两千年来，以道德代替法制，至明代而极，这就是一切问题的症结。

    文官摄政的一大原因是以前各个王朝，凡君主年幼，必定有他的叔父、堂兄这样的人物代为摄政，而这恰恰为本朝所不能容许。按照规定，所有皇室的支系，包括皇帝的叔父、兄弟以至除皇太子以外的儿子，一到成年就应当离开京城到自己的封地。
